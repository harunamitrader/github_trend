AI Tools Others Update: Claude Opus 4.7 for Finance, Gemini API Webhooks, and ChatGPT Memory Sources

汎用 AI ツールの 2026年5月初週のアップデートをまとめました。金融特化型モデル Claude Opus 4.7 の発表、Gemini API の Webhooks 対応、ChatGPT のパーソナライズ機能強化など、利便性と実用性を高める更新が続いています。

リリース日 2026-05-06
カテゴリー Others
ツール Claude, Gemini, ChatGPT, NotebookLM, Obsidian

主な変更点

2026年5月初週、主要な AI サービスに重要な新機能やインテグレーションが追加されました。ビジネス活用を加速させる機能から、個人の使い勝手を向上させる改善まで多岐にわたります。

  • Claude Opus 4.7 (May 5): 金融サービス向けに特化した新モデル。市場調査、財務モデル構築、信用報告書のドラフト作成などで高い性能を発揮します。
  • Gemini API Webhooks (May 4): イベント駆動型の Webhooks を導入。Batch API や長時間実行されるタスクの完了を、ポーリングなしでリアルタイムに受信可能に。
  • Gemini API File Search: マルチモーダル検索、カスタムメタデータフィルタ、ページレベルの引用に対応。
  • ChatGPT Memory Sources: 回答に影響を与えた過去のチャットやファイルなどの情報源を確認・編集できる機能を追加。Gmail 連携によるパーソナライズも強化。
  • NotebookLM: 18歳以上の学生向けに Google Classroom 連携を開放。講義資料から自分専用のノートブックを構築可能に。
  • Obsidian: モバイル版 2.0 の洗練、プラグインの API キーを安全に管理するための「Keychain」機能を導入。

影響や使いどころ

Claude Opus 4.7 は金融業界のプロフェッショナルにとって強力な武器となります。Gemini API の Webhooks は開発者のワークフローを大幅に簡略化し、コスト削減にも寄与します。ChatGPT の Memory Sources は、AI がなぜその回答をしたかの透明性を高め、ユーザーによる制御を容易にします。

まとめ

各社とも、特定の専門領域（金融など）への特化と、既存エコシステム（Google Workspace, Microsoft Teams 等）との深い統合、そして情報の透明性確保へと舵を切っています。
