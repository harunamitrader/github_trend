Claude Code v2.1.92: Bedrock 導線と /cost 可視化を強化

Bedrock 初期設定の対話ウィザード追加と、/cost の内訳可視化、/release-notes の操作改善が入った更新です。

リリース日 2026-04-04
バージョン v2.1.92
ツール Claude Code

主な変更点

Anthropic の公式 GitHub Releases では、v2.1.92 が 2026-04-04 00:42 UTC に公開されました。今回の追加点では、ログイン画面から Bedrock を選んだ際に対話式セットアップウィザードが起動し、AWS 認証、リージョン設定、資格情報確認、モデル固定まで順に進められます。

あわせて /cost にモデル別と cache hit の内訳が表示されるようになり、subscription 運用時のコスト確認が実用的になりました。/release-notes も対話式のバージョン picker へ変わっています。

影響や使いどころ

運用面では、forceRemoteSettingsRefresh により起動時に管理設定の最新取得を強制でき、取得失敗時は fail-closed で止められます。チーム管理を厳しくしたい環境向けの変更です。

加えて、tmux 環境の subagent 失敗、JSON 文字列化された tool 入力、Homebrew 更新案内、Linux sandbox の apply-seccomp など、長時間運用で効く修正がまとまっています。

まとめ

v2.1.92 は新機能と運用改善のバランスがよい更新です。Bedrock 導入を簡単にしつつ、コスト可視化と管理設定の厳格運用まで押さえています。
