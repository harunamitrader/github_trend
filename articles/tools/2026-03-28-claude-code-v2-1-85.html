Claude Code v2.1.85: MCP 制御の強化とフック機能の拡張

MCP サーバー接続用の動的環境変数と、フックの条件付き実行に対応。

リリース日 2026-03-27
バージョン v2.1.85
ツール Claude Code

主な変更点

MCP (Model Context Protocol) の柔軟な運用

MCP サーバーの設定に新しい環境変数（CLAUDE_CODE_MCP_SERVER_NAME / CLAUDE_CODE_MCP_SERVER_URL）が導入されました。これにより、動的なサーバー構築やコンテナ環境からの接続がより簡単になります。

フック機能の条件付き実行

config.json 内のフック設定に if フィールドが追加されました。特定の条件下（例：OSの種類や特定の環境変数の有無）でのみコマンドを実行することが可能になり、プロジェクトごとのカスタマイズ性が向上しています。

UX と接続性の改善

  • 長期タスクの可視化: /loop 等のタスク実行時にタイムスタンプマーカーが表示されるようになり、進捗確認が容易になりました。
  • URL 長制限の緩和: ディープリンクのクエリ長制限が 5,000 文字に引き上げられ、より複雑なコンテキストの受け渡しが可能になりました。

まとめ

今回のアップデートは、MCP を活用した外部ツール連携の自由度をさらに高める内容となっています。特に条件付きフックは、チーム開発において環境ごとの微調整を行う際に非常に強力な武器となるでしょう。
