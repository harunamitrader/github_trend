Cursor 3.0: Agents Window と Design Mode を導入

複数エージェントを並列運用する新 UI と、ブラウザ上で UI 要素を直接指定できる Design Mode が追加されました。

リリース日 2026-04-02
バージョン 3.0
ツール Cursor

主な変更点

Cursor の公式 changelog では、2026-04-02 付で Cursor 3.0 が公開されています。中心は Agents Window で、ローカル、worktree、クラウド、remote SSH をまたいで複数エージェントを並列実行できる新しい操作面です。

同じ画面内で Design Mode も使えるようになり、ブラウザ上の UI 要素を直接指定してエージェントへ渡せます。UI 修正時の「どこを直すか」を会話だけで説明する負荷を下げる更新です。

実務で効くポイント

Editor 側では /worktree と /best-of-n が新コマンドとして整理され、複数モデルの並列比較を isolated worktree 前提で回しやすくなりました。加えて MCP Apps の structured content 対応、Explorer subagent 起動のキャッシュ、shared chats への plan 同梱など、エージェント運用を前提にした細部が更新されています。

まとめ

Cursor 3.0 は単なる見た目変更ではなく、エージェント中心のワークスペースへ軸足を移す大きな区切りです。複数タスクの並列実行や UI 指示の明確化を重視するチームほど恩恵が大きい更新です。
