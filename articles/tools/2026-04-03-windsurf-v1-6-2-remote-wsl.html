Windsurf v1.6.2: Remote-WSL 修正と軽微な UX 改善

2026-04-03 の公式 changelog で、Remote-WSL 拡張の修正と小規模な UX 調整が反映されました。

リリース日 2026-04-03
バージョン v1.6.2
ツール Windsurf

主な変更点

Windsurf の公式 changelog では、1.6.2 が 2026-04-03 付で公開されています。今回の記載は短く、Remote - WSL 拡張の修正と minor UX fixes の 2 点が更新対象です。

影響や使いどころ

大きな新機能追加ではありませんが、WSL ベースの開発環境を使っている場合は追従価値があります。前日の 1.6.1 で Deploys や Memories 周りの追加が入った直後なので、その後続の安定化パッチという位置づけです。

UI 面も軽微に整えられているため、リモート環境を常用するユーザーは優先度を下げずに見ておきたい更新です。

まとめ

v1.6.2 は派手さはないものの、Remote-WSL 利用時の不具合を潰すメンテナンスリリースです。リモート開発中心のチームでは反映しておくのが無難です。
