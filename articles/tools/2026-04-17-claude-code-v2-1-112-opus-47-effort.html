Claude Code v2.1.112: Opus 4.7 xhigh と /ultrareview を追加

Opus 4.7 向けの xhigh effort、対話式 /effort、クラウドでの包括レビュー、PowerShell tool の展開が入った更新です。

確認日 2026-04-17
バージョン v2.1.112
ツール Claude Code

主な変更点

Anthropic の Claude Code changelog では、v2.1.112 が最新として公開されています。v2.1.112 は auto mode で Opus 4.7 が一時的に利用できない問題の修正が中心で、直前の v2.1.111 で入った Opus 4.7 関連の機能を安定化する位置づけです。

  • Opus 4.7 xhigh: Opus 4.7 向けに xhigh effort が追加され、/effort、--effort、モデルピッカーから選べます。
  • 対話式 /effort: 引数なしで /effort を呼ぶと、矢印キーで調整できるスライダーが開くようになりました。
  • /ultrareview: クラウド上の並列マルチエージェント分析を使い、現在のブランチや指定 PR を包括的にレビューできます。
  • PowerShell tool: Windows 向け PowerShell tool が段階的に展開され、環境変数で有効化・無効化できます。

影響や使いどころ

今回の更新は、モデル選択とレビュー実行をより細かく制御したいユーザーに効きます。特に xhigh は、速度と推論深度のバランスを高めたい長めの設計・修正作業で使いやすく、/ultrareview は PR レビュー前の自己点検に向いています。

補足

v2.1.111 では、Auto mode の要件緩和、terminal の dark/light に合わせる theme、権限プロンプトを減らす /less-permission-prompts skill、Windows の環境ファイル適用修正なども含まれています。

まとめ

v2.1.112 は小さな修正リリースですが、v2.1.111 の大型変更と合わせて見ると、Claude Code のモデル制御、レビュー、Windows 運用の実用性を押し上げる更新です。
