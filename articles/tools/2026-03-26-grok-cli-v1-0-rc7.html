Grok CLI v1.0.0-rc7: デーモン実行と強力なサンドボックス機能

定時実行の自動化や新しいサンドボックス、さらに Telegram 連携を大幅に強化。

リリース日 2026-03-26
バージョン v1.0.0-rc7
ツール Grok CLI

主な変更点

デーモンによる定時実行（Scheduled Runs）

エージェントをバックグラウンドでデーモンとして稼働させる、ヘッドレス実行機能が追加されました。特定の時刻や周期で自動的にタスクを実行し、情報を収集・通知することが可能になっています。

Shuru Sandbox Mode の導入

よりセキュアなコード実行環境を提供するサンドボックスモードが導入されました。ネットワークアクセス、ポート、ファイル権限などを細かく制限しつつ、AI にシェルコマンドを実行させることができます。

Telegram ブリッジの進化

RC6 以降、Telegram 連携が劇的に強化されています。

  • telegram_send_file ツールの追加。
  • whisper.cpp を用いたボイスメッセージのローカル文字起こし。
  • Vision サブエージェントによる、OCR を含む高度な画像解析の自動化。
  • 外部ブリッジ設定による、外出先からのエージェント制御。

全体的なバグ修正とリファインド

CLI 出力のカラーリング改善、エージェント生成タスクの安定性向上、そして内部ロギング機能の拡充が含まれています。

まとめ

Grok CLI v1.0.0-rc7 は、エージェントを「手動で動かす」段階から「バックグラウンドで自動監視・自動処理させる」段階へと進化させたリリースです。特に Telegram をハブとした、サーバーレスに近い自律運用の道筋が示されました。
