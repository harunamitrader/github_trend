Cline v3.77.0: Lazy Teammate Mode と chunked read_file を追加

実験的モード追加とファイル読み込み改善を中心に、Kanban 周辺の細かな修正も入った最新版です。

リリース日 2026-04-01
バージョン v3.77.0
ツール Cline

主な変更点

公式 GitHub Releases によると、v3.77.0 では実験機能として Lazy Teammate Mode が追加されました。名称どおり、分業や補助的なエージェント運用を意識したトグルで、既存の Kanban 系フローを補完する位置づけに見えます。

もう一つの追加点は、read_file ツールの chunked reading 対応です。大きいファイルを丸ごと読むのではなく、必要な断片へ寄せた読み出しがしやすくなり、コンテキスト消費や読み込みコストの圧縮に効きます。

補助的な修正

修正項目では、yolo / headless モードで new_task を system prompt から除外する変更と、Kanban デモ動画の整形が含まれます。さらに Notification フックの磨き込みも入っており、周辺挙動の安定化を進めたリリースです。

まとめ

v3.77.0 は大型機能よりも、分業補助の実験機能と読み込み効率化を積み増した更新です。最近の Kanban 路線を追っているユーザーには自然な次の一歩です。
