Gemini CLI nightly 20260402: 起動処理と subagent 設定反映を改善

2026-04-02 の nightly では、起動クリーンアップの並列化や subagent 設定の即時反映など、日常運用に近い改善が入りました。

リリース日 2026-04-02
バージョン v0.36.0-nightly.20260402
ツール Gemini CLI

主な変更点

Google の公式 GitHub Releases では、v0.36.0-nightly.20260402.13ccc1645 が 2026-04-02 04:33 UTC に公開されました。今回の nightly では、subagent ツール更新時に configuration overrides を即時反映する修正が先頭に入っています。

加えて、起動時 cleanup を並列化してバックグラウンドで処理する改善、allow always と path 付きコマンドの扱いの修正、terminal escape sequence の漏れ防止など、CLI の実運用に近い箇所が細かく磨かれています。

影響や使いどころ

Nightly を追うチームにとっては、subagent 設定の即時反映と startup cleanup の高速化が実務上の差分です。加えて、skills の接頭辞処理や redirect 許可のサポート修正など、複雑なワークフローで効きやすい変更も含まれます。

まとめ

今回の nightly は、派手な機能追加よりも、subagent・権限・端末表示の挙動を整えるメンテナンス寄りの更新です。nightly チャンネルを監視対象にしているなら反映して問題ありません。
