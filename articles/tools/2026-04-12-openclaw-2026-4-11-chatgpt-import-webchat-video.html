OpenClaw 2026.4.11: ChatGPT import と Webchat / video generation を強化

Dreaming の ChatGPT import、Webchat の構造化表示、video generation の providerOptions などをまとめた運用改善リリースです。

リリース日 2026-04-12
バージョン v2026.4.11
ツール OpenClaw

主な変更点

GitHub Releases では、2026-04-12 付で OpenClaw 2026.4.11 が Latest として公開されています。今回の中心は、Dreaming / memory-wiki に ChatGPT import ingestion を追加し、Imported Insights と Memory Palace のサブタブから取り込んだ会話や wiki ページを確認できるようにした点です。

影響や使いどころ

Control UI / webchat では、assistant の media、reply、voice directive を構造化された chat bubble として表示し、[embed ...] によるリッチ出力も扱えるようになっています。Webchat を OpenClaw の主画面として使う場合、生成物や音声付き応答を会話履歴と結びつけて見やすくする更新です。

補足

video generation では URL-only delivery、typed providerOptions、reference audio input、adaptive aspect-ratio などが追加され、動画プロバイダーごとの表現力を出しやすくなっています。修正面では OpenAI / Codex OAuth の scope 処理、音声 transcription、Google Veo の unsupported field 送信などが整理されています。

まとめ

2026.4.11 は、OpenClaw の記憶取り込み、Webchat 表示、メディア生成、OAuth / provider 周辺をまとめて安定化する更新です。2026.4.10 の Codex provider / Active Memory に続き、長期利用とマルチメディア運用を現実的にする方向のリリースです。
