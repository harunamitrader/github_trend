Antigravity v1.21.7: Firebase とのフルスタック統合

Google AI Studio で、プロンプトひとつでフルスタックアプリが完結する「Vibe Coding」が加速。

リリース日 2026-03-28
バージョン v1.21.7 / v1.21.6
ツール Antigravity

主な変更点 (v1.21.6 - 1.21.7)

Firebase 統合とフルスタック開発の自動化

Google AI Studio と Antigravity の連携が飛躍的に進化しました。Firebase をネイティブに統合したことで、フロントエンドの生成からバックエンドの構築（認証、データベース、インフラ）までを、AI が対話形式で完結させる「Vibe Coding」が正式にサポートされました。

AGENTS.md によるルール管理の公式化

これまでの GEMINI.md に加え、AGENTS.md からのカスタム・ルール読み込みが公式にサポートされました。プロジェクト固有のコーディング規約やエージェントの振る舞いを、標準的な形式で管理できるようになっています。

Linux Sandboxing の導入

Antigravity Terminal において、Linux 環境でのサンドボックス化がサポートされました。これにより、開発環境の分離と安全性が大幅に強化されています。

UI/UX の刷新

  • シンプルなチャット画面: 必要な情報をより凝縮した、モダンでフラットなデザインに刷新。
  • サイドバーの改善: ワークスペースの切り替えやアーカイブ機能がより直感的に。
  • ワンクリック・アーカイブ: 会話の保存と整理がこれまで以上にスムーズに行えるようになりました。

まとめ

Antigravity は、単なるコーディングアシスタントから、Google Cloud エコシステム（特に Firebase）を自在に操る「自律型エンジニア」へと、その立ち位置を確立しつつあります。特に v1.21.7 における Firebase との統合は、アプリケーション開発における概念そのものを覆す大きな一歩となるでしょう。
