Release Notes

Google Antigravity: Firebase 統合によるプロダクション級アプリの「Vibe Coding」実現

コーディングエージェントに Firebase のフルパワーを授け、認証・DB・インフラの自動構築が可能に。

リリース日 2026-03-26
新機能 Firebase Integration
ツール Antigravity

主な変更点

Firebase とのネイティブ統合

Google AI Studio 内のコーディングエージェント「Antigravity」が Firebase と直接連携できるようになりました。これにより、エージェントがユーザーの代理としてアプリのバックエンド（Firebase Auth, Firestore 等）を自動構成し、展開することが可能になります。

Vibe Coding プロセスの一般公開

「雰囲気（Vibe）で語るだけで動作する完全なアプリを作る」体験が、誰でもプレビューできるようになりました。AI がフロントエンドだけでなく、サーバーサイドのインフラ（Google Cloud Functions for Firebase）まで、一貫して管理・修正します。

強固なセキュリティと API 管理

秘匿性の高い API キーの安全な保管場所と、プロジェクトごとのアクセス権限管理が Google AI Studio 側で統合。マルチプレイヤー（複数人共同開発）対応も強化され、個人開発からチーム開発までカバーします。

まとめ

今回の Firebase 統合により、Antigravity は「プロトタイプを作るツール」から「実際に動作する本番品質のプロダクトを作るツール」へと進化しました。Google エコシステムならではの、AI とクラウドインフラの究極のシームレス体験が始まろうとしています。
