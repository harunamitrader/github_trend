Windsurf v1.9600.41: Adaptive Fix

Adaptive model router で最初のリクエスト後にモデル切り替えできなくなる不具合が修正されました。

リリース日 2026-04-07
バージョン v1.9600.41
ツール Windsurf

主な変更点

Windsurf の公式 changelog では、2026-04-07 付で v1.9600.41 が公開されています。内容は大きな機能追加ではなく、Adaptive model router の挙動修正です。最初のリクエスト後にモデルを切り替えられなくなるバグが直されました。

影響や使いどころ

Adaptive を試しながら、途中でより高性能なモデルや別の選択肢へ切り替えたい場面は実務で普通にあります。その操作が止まっていた問題なので、影響範囲は見た目以上に大きめです。Adaptive を運用に入れ始めたユーザーほど更新優先度は高いです。

補足

公式 changelog では、影響を受けたユーザーに対して quota reset と overage restore を行ったことも明記されています。単なるバグ修正に加え、課金影響のリカバリーまで含めて対処した形です。

まとめ

v1.9600.41 は小粒ですが重要です。Adaptive を安心して使い続けるための修正版として見てよい更新です。
