Gemini CLI v0.38.1: stable パッチリリースを公開

v0.38.0 系からの cherry-pick を反映した stable パッチで、GitHub Releases 上の latest タグが v0.38.1 に更新されました。

リリース日 2026-04-15
バージョン v0.38.1
ツール Gemini CLI

主な変更点

Google Gemini CLI の GitHub Releases では、2026-04-15 付で v0.38.1 が latest として公開されています。リリースノート上では、release/v0.38.0 系へ cherry-pick したパッチを反映して v0.38.1 を作成した更新として説明されています。

影響や使いどころ

大きな機能追加ではなく、stable 系の修正を取り込むためのパッチです。preview や nightly を追わずに安定版を使っている場合は、v0.37.2 から v0.38.1 へ latest stable が進んだ点を確認しておくとよいです。

補足

同じリリース一覧では、2026-04-15 付の nightly として v0.40.0-nightly.20260415.g06e7621b2 も確認できます。nightly 側では plan prompt、MCP error handling、AskUser の mouse click 対応など、多数の開発中変更が含まれています。

まとめ

v0.38.1 は、Gemini CLI の stable 利用者向けに最新の修正を取り込む小さな更新です。運用環境では latest タグが変わっている点を見落とさないようにしたいリリースです。
