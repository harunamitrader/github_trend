Release Notes

GitHub Copilot 3月26日アップデート: マージコンフリクト解法支援と新ダッシュボード

プルリクエストのコンフリクト解消を Copilot がサポート開始。エージェントのアクティビティ追跡も強化。

リリース日 2026-03-26
バージョン Mar. 26 Update
ツール GitHub Copilot

主な変更点

マージコンフリクト解消支援 (Preview)

プルリクエスト（PR）のコメント欄から @copilot を呼び出し、マージコンフリクトの直接的な解消を依頼できるようになりました。提案されたコードをレビューし、そのまま採用することで煩雑なマージ作業が劇的に短縮されます。

新しいプルリクエスト・ダッシュボード

自分に関連する PR 全体を俯瞰し、優先順位を整理するための新ダッシュボードがパブリックプレビューとして公開されました。

AI エージェントのアクティビティ可視化

Issues や Projects において、AI Coding エージェントが実行したアクションがタイムライン上で明確に区別して表示されるようになりました。これにより、人間と AI の協調プロセスがより透明になります。

モデルラインナップの整理

  • Gemini 3 Pro のサポート終了（Retired）と最新モデルへの置き換え。
  • GitHub Actions のサマリーから、エージェントが構成したワークフローを直接確認できるように改善。
  • Raycast 等の周辺ツールを用いたエージェントログのライブ監視機能（Preview）の整備。

まとめ

GitHub Copilot の 3 月末のアップデートは、「PR コンフリクト」という開発現場の痛点に対する直接的な解決策を提示した非常に強力な内容です。また、エージェントのアクティビティ可視化は、単なるコード補完を超えた「コーディングの自動化基盤」としての GitHub の進化を象徴しています。
