Claude Code v2.1.79: Console 認証対応と信頼性の強化

Anthropic Console 経由のログイン対応と、長時間セッションにおけるハングアップの解消。

リリース日 2026-03-18
バージョン v2.1.79
ツール Claude Code

主な新機能

Claude Code v2.1.79 では、法人利用や API 利用者にとって待望の認証機能が追加されました。

Anthropic Console 認証への対応

`claude auth login --console` フラグが追加され、Anthropic Console (API クレジット決済) 経由での認証が可能になりました。 これにより、Claude.ai の個人プランとは異なる API アカウントでの利用が容易になります。

実行時間の可視化

`/config` メニューに「ターンの所要時間を表示（Show turn duration）」の切り替えトグルが追加されました。 各応答にどれくらいの時間がかかったかを計測し、効率化の指標とすることができます。

改善と信頼性の向上

  • API タイムアウトの改善: 非ストリーミング応答時に、1リクエストにつき 2 分間のタイムアウトを設定。長時間応答を待ってセッションが固まる問題を防止します。
  • `-p` モードの修正: 標準入力を指定せずにサブプロセスとして起動した際にハングアップする問題を修正。CI 環境等での動作が安定しました。
  • メモリ使用量の最適化: 起動時のメモリ消費量を約 18MB 削減。より軽量に動作するようになりました。
  • プラグインの拡張: `CLAUDE_CODE_PLUGIN_SEED_DIR` において、セミコロン区切りで複数のシードディレクトリを指定可能に。

まとめ

v2.1.79 は、地味ながらも堅実な改善が詰め込まれたアップデートです。 Console 認証の導入により、ビジネスシーンでの Claude Code 活用がさらに加速することでしょう。
