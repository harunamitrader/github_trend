Cline v3.76.0: 看板管理の統合とループ検出による安全性の向上

人気の Kanban 機能を標準搭載し、トークンの浪費を防ぐ「ツール実行ループ検出」を導入。

リリース日 2026-03-27
バージョン v3.76.0
ツール Cline

主な変更点

Cline Kanban の本格統合

Webview 内に Cline Kanban ローンチモーダルが統合されました。CLI から起動する際も、デフォルトで Kanban が有効になるよう変更されています。タスク進捗を視覚的に管理したいという要望に強力に応える内容です。

ツール実行・無限ループの自動検出と停止

エージェントが同じツールの呼び出しを永久に繰り返すことを防ぐ、新しいガードレール機能が追加されました。これにより、意図しない高額なトークン消費のリスクを大幅に削減できます。

UI の改善とバグ修正

  • チャット内の各種ヒント表示（ヒント・カード）をトグルで無効化できるオプションを追加。
  • Windows 環境において npx.cmd を介して Kanban を正しく起動できるよう修正。
  • Mac 上でのターミナル出力の視認性改善を含む、複数のデザイン調整を実施。

まとめ

Cline v3.76.0 は、「使い勝手（Kanban）」と「コスト・安全性（ループ検出）」の両面が大幅に進化したリリースです。特に、長時間エージェントを自律稼働させる際のループ監視は、実務での信頼性を一段引き上げる重要な機能となるでしょう。
