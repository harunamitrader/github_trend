Cline v3.76.0: 看板管理の統合とループ検出による安全性の向上

人気の Kanban 機能を標準搭載し、トークンの浪費を防ぐ「ツール実行ループ検出」を導入。

記事作成日 2026-03-27
作成時順位 999位
対象 repo cline/cline

これは何か

VS Code 拡張機能として人気の自律型コーディング助手「Cline」の最新アップデートです。タスク管理を効率化する「Kanban（看板）」機能の本格統合と、エージェントの暴走を防ぐためのガードレール機能が追加されました。

何ができるか

新しく Cline Kanban ローンチモーダルが Webview 内に追加され、CLI 版でもデフォルトで Kanban が起動されるようになりました。これにより、進行中のタスクを視覚的に把握しやすくなります。また、技術的な面では「ツール実行ループ検出機能」が実装され、同じツール呼び出しを繰り返す無限ループを検知して停止させることが可能になりました。

目立つポイント

ループ検出機能は、開発者が意図しないトークンの大量消費（＝コストの急増）を防ぐための極めて重要なアップデートです。また、チャット内の各種ヒント表示を無効化するトグルも追加され、よりクリーンな UI で作業に集中できるようになりました。

セットアップや使い方の流れ

VS Code の拡張機能マーケットプレイスから更新可能です。Kanban 機能を使用するには、新しく追加されたローンチモーダルからアクセスします。Windows 環境で npx.cmd 経由で Kanban が起動しない問題も修正されており、OS を問わず安定して動作します。

どんな人向けか

複数のタスクを並行して進める際に進捗を視覚的に管理したい開発者や、エージェントを自律的に長時間動かす際の予期せぬコスト変動を懸念しているユーザーに特にお勧めです。

注意点

Kanban への移行に伴い、CLI 版では従来のビューからの移行が必要になる場合があります。ループ検出が作動した際は、プロンプトやタスクの指示を微調整してループを解消する必要があります。

まとめ

v3.76.0 は、Cline を単なるコーディングツールから「タスク管理を含めた総合的な開発プラットフォーム」へと昇華させるアップデートです。安全性（ループ検出）と視認性（Kanban）の両面が強化され、より実務投入しやすいツールへと進化を遂げました。
