OpenClaw v2026.4.25: Voice/TTS の大幅強化と OpenTelemetry 対応

Azure Speech や ElevenLabs v3 への対応、プラグイン管理の決定論的刷新、OpenTelemetry によるフルスタック監視など、プロダクション運用に向けた安定性が向上。

リリース日 2026-04-27
バージョン v2026.4.25
ツール OpenClaw

主な変更点

自律型エージェント基盤 OpenClaw の最新版 v2026.4.25 がリリースされました。音声合成エンジンの拡充と、運用監視のためのインフラ強化が中心となっています。

  • Voice/TTS エコシステムの拡充: Azure Speech、ElevenLabs v3、Xiaomi、Inworld など、複数の外部 TTS プロバイダーにネイティブ対応しました。チャットごとの自動 TTS 制御も可能になっています。
  • Cold Registry の導入: プラグインのインストールパスやメタデータを管理するレジストリを刷新。アップデート時の決定論的な挙動と効率性が大幅に向上しました。
  • OpenTelemetry 対応: モデル呼び出し、ツール実行ループ、メモリ管理の各レイヤーで詳細なテレメトリ取得が可能になり、大規模運用のデバッグが容易になりました。
  • ブラウザ自動化の安全強化: ブラウザ操作時のサンドボックス制限と、視覚的な実行確認 UI の改善が行われました。

影響や使いどころ

多様な音声エンジンのサポートにより、より自然でコンテキストに沿った音声対話型エージェントの構築が容易になります。また、OpenTelemetry への対応により、企業導入時などのオブザーバビリティ（可視性）要件を満たしやすくなりました。

まとめ

OpenClaw は単なる実験的なツールから、高い信頼性と監視機能を備えた、実用的なエージェント・プラットフォームへと進化を続けています。
