Claude Code v2.1.84: PowerShell 対応と制御機能の拡充

Windows ユーザー待望の PowerShell ツール (Preview) と、環境変数による詳細な挙動制御を導入。

記事作成日 2026-03-27
作成時順位 999位
対象 repo anthropic-ai/claude-code

これは何か

Anthropic のターミナル用 AI 開発エージェント「Claude Code」のマイナーアップデートです。Windows 環境での操作性を劇的に向上させる新ツールの導入や、エンタープライズ利用を想定した管理機能、そして数多くの UX 改善が含まれています。

何ができるか

最大の新機能は、Windows 向けの PowerShell ツール です。現在はオプトインのプレビュー版として提供されており、Windows ユーザーは bash エミュレーションを介さずにネイティブなシェル操作を Claude に任せることが可能になります。また、ANTHROPIC_DEFAULT_*_MODEL_SUPPORTS などの環境変数が追加され、Vertex AI や Bedrock などのサードパーティ・プロバイダーを使用する際の思考能力（Thinking/Effort）の検出を明示的に制御できるようになりました。

目立つポイント

「アイドル・リターン・プロンプト」の導入により、75分以上放置されたセッションに戻った際、Claude が /clear を推奨するようになりました。これにより、不必要なトークンの再キャッシュを防ぎ、コストを抑えることができます。また、ディープリンク（claude-cli://）が標準のターミナル以外でも正しく開くようになったほか、CJK 入力（日本語入力）時のカーソル追従などのアクセシビリティも改善されています。

セットアップや使い方の流れ

通常通り npm install -g @anthropic-ai/claude-code でアップデート可能です。PowerShell ツールを試すには、公式ドキュメントに従って設定を有効化する必要があります。環境変数の設定により、出力のカスタマイズやタイムアウトの閾値（CLAUDE_STREAM_IDLE_TIMEOUT_MS）の調整も可能です。

どんな人向けか

特に Windows 上で Claude Code を活用している開発者や、プロンプトキャッシュを効率化してコストを抑えたいパワーユーザー、さらにはサードパーティ API を介して Claude を運用しているチームに適しています。

注意点

PowerShell ツールはまだプレビュー段階のため、複雑なスクリプトの実行には注意が必要です。また、MCP ツールの説明文が 2KB に制限されるようになったため、大規模な MCP サーバーを使用している場合は影響を受ける可能性があります。

まとめ

v2.1.84 は、ツールの安定性と Windows 対応、そしてコスト管理という実用面にフォーカスした堅実なバグフィックス＆機能追加アップデートです。ネイティブな PowerShell サポートの開始は、Windows 開発者にとって大きな前進と言えるでしょう。
