Claude Code v2.1.84: PowerShell 対応と制御機能の拡充

Windows ユーザー待望の PowerShell ツール (Preview) と、環境変数による詳細な挙動制御を導入。

リリース日 2026-03-27
バージョン v2.1.84
ツール Claude Code

主な変更点

PowerShell ツールの導入 (Preview)

Windows 環境において、Claude が直接 PowerShell を実行できる新しいツールがオプトインのプレビュー版として追加されました。これにより、Windows ネイティブな環境での開発作業がよりスムーズになります。

高度なモデル・プロバイダー制御

ANTHROPIC_DEFAULT_*_MODEL_SUPPORTS 系の環境変数が追加され、Vertex AI や Bedrock などのサードパーティ経由で Claude を使用する際、Thinking（思考能力）や Effort（工程の重さ）の検出を明示的に指定できるようになりました。

アイドル時のコスト最適化

セッションを 75 分以上放置して戻った際、Claude が /clear コマンドの実行を推奨する「アイドル・リターン・プロンプト」が導入されました。これにより、不要なコンテキストの再送によるトークン消費を抑制できます。

UX とアクセシビリティの向上

  • 日本語などの CJK 入力時にカーソルが正しく追従するように改善。
  • claude-cli:// ディープリンクが標準以外のターミナルでも正しく動作。
  • MCP ツールの説明文を 2KB に制限し、リクエストの安定性を向上。
  • セッション開始時の信頼性向上と、各種マイナーバグの修正。

まとめ

今回の v2.1.84 は、Windows ユーザーへの対応強化と、エンタープライズやサードパーティ API 利用時の制御性に焦点を当てた実用的なアップデートです。特にアイドル時のクリア推奨は、長時間にわたる開発セッションでのコスト管理に大きく寄与するでしょう。
