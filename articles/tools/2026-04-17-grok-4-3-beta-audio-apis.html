Grok 4.3 Beta & 新オーディオ API: スライド生成と Tesla 譲りの音声合成

ネイティブなスライド生成機能を備えた Grok 4.3 Beta と、高性能な独立オーディオ API が登場。

リリース日 2026-04-17
バージョン v4.3 Beta / Audio API
ツール Grok

主な変更点

xAI は、Grok シリーズの最新アップデートとして、モデルの進化と独立した開発者向け API の拡充を発表しました。

  • Grok 4.3 Beta: 早期アクセス「SuperGrok Heavy」プラン向けに公開。待望のネイティブ・スライド資料生成機能が追加されました。
  • Audio API: Speech-to-Text (STT) および Text-to-Speech (TTS) の独立した API をリリース。多言語対応、スピーカー認識（Diarization）、構造化出力に対応しています。
  • Tesla 技術の転用: これらのオーディオエンジンは、Tesla 車両や Starlink のサポート窓口で使用されている実戦的な技術をベースに構築されています。

影響や使いどころ

スライド生成機能のネイティブ搭載により、リサーチからプレゼン資料の作成までを Grok 内で完結できるようになります。また、オーディオ API は極めて堅牢で高速に動作するように設計されており、モバイルアプリや IVR（電話自動応答）システムへの統合で高いパフォーマンスを発揮します。多言語に対応しているため、グローバルなカスタマーサービスボットなどの構築にも適しています。

まとめ

Grok は、X のエコシステムを超えた、実用的かつ汎用的な AI インフラとしての性格を強めています。特に、既存の Musk 系各社のハードウェア/インフラで鍛えられた音声技術を API 化した点は、他の LLM ベンダーに対する強力な差別化要因となるでしょう。
