Claude Code v2.1.90: /powerup と PowerShell 権限制御を強化

対話型レッスン /powerup の追加に加え、PowerShell 権限チェックと長時間セッション性能の改善が入った更新です。

リリース日 2026-04-02
バージョン v2.1.90
ツール Claude Code

主な変更点

Anthropic の公式 GitHub Releases では、v2.1.90 が 2026-04-01 23:41 UTC に公開されました。今回の目玉は /powerup で、Claude Code の機能をアニメーション付きの対話型レッスンで学べるようになっています。

運用面では、CLAUDE_CODE_PLUGIN_KEEP_MARKETPLACE_ON_FAILURE が追加され、git pull に失敗しても既存 marketplace キャッシュを保持できるようになりました。オフラインや不安定なネットワーク環境での plugin 利用に効く変更です。

実務で効くポイント

PowerShell 周りでは、末尾の & を使ったバックグラウンド化、-ErrorAction Break でのハング、アーカイブ展開時の TOCTOU など、権限チェックの抜け道になり得る箇所がまとめて補強されました。Windows 利用者には優先度が高い修正です。

そのほか、長い会話で transcript 書き込みが二乗的に遅くなる問題、巨大な SSE フレーム処理、--resume の一覧表示など、継続セッションの体感速度に関わる改善も入っています。

まとめ

v2.1.90 は、学習導線を増やしつつ、PowerShell の安全性と長時間利用時の安定性を底上げするリリースです。特に Windows で本番運用しているなら追従価値があります。
