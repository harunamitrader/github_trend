Gemini CLI v0.42.0-nightly: GrepTool のケースセンシティビティ修正とメトリクス整合性向上

検索ツールの信頼性向上と、内部統計処理の標準化を主眼としたメンテナンスアップデート。

リリース日 2026-05-04
バージョン v0.42.0-nightly
ツール Gemini CLI

主な変更点

今回のアップデートでは、コード検索ツール(GrepTool)の挙動修正と、内部統計の整合性向上が主な内容となっています。

  • GrepTool の修正: 大文字小文字の区別（Case-sensitivity）に関する挙動を修正し、検索の精度と一貫性を向上させました。
  • メトリクスの整合性向上: アクション実行数やトークン使用量などのメトリクス計算ロジックを見直し、統計データの正確性を向上させました。
  • レポーティングの標準化: ツール実行結果の出力形式を標準化し、上位のエージェントや監視ツールによる解析を容易にしました。

影響や使いどころ

コードベースの広域検索において、意図しない検索漏れが抑制されるため、リファクタリングや大規模なコード解析の信頼性が向上します。また、メトリクスの正確な把握は、コスト管理やパフォーマンスチューニングにおいて重要です。

まとめ

地味ながらもツールの「足回り」を固める重要なアップデートです。特に大規模なプロジェクトで GrepTool を多用する環境では、早めの適用を推奨します。
