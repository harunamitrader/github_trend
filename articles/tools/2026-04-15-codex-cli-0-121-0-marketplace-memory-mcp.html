Codex CLI 0.121.0: marketplace、memory、MCP/plugin を強化

plugin marketplace の追加、TUI prompt history、memory mode 管理、MCP Apps tool calls などが入った Codex CLI 更新です。

リリース日 2026-04-15
バージョン 0.121.0
ツール Codex

主な変更点

OpenAI の公式 Codex changelog では、2026-04-15 付で Codex CLI 0.121.0 が公開されています。今回の中心は、CLI / TUI / app-server / MCP / plugin 周辺の運用機能強化です。

  • marketplace add: codex marketplace add と app-server が、GitHub、git URL、ローカルディレクトリ、直接の marketplace.json URL から plugin marketplace を追加できるようになりました。
  • TUI prompt history: Ctrl+R の reverse search と、受理済み slash command の local recall が追加されています。
  • memory 管理: TUI と app-server に memory mode、memory reset / deletion、memory-extension cleanup の操作が入りました。
  • MCP/plugin 拡張: MCP Apps tool calls、namespaced MCP registration、parallel-call opt-in、MCP server の sandbox-state metadata などが追加されています。

影響や使いどころ

plugin marketplace と memory 管理が入ったことで、Codex を単発 CLI ではなく、継続的な開発環境として運用しやすくなります。MCP/plugin の名前空間や並列呼び出し設定は、複数サーバーを使う構成で衝突や意図しない動作を減らす助けになります。

補足

バグ修正では、macOS sandbox/proxy、Windows の cwd/session matching、rate-limit/account handling、Guardian timeout、app-server の thread unload、MCP/tool-call の edge case などが対象です。

まとめ

0.121.0 は、Codex の拡張基盤と長期利用の管理機能を一段進めるリリースです。plugin、MCP、memory を組み合わせて使っているユーザーほど確認しておきたい更新です。
