Tool Update

Claude Code v2.1.121: MCP alwaysLoad と /skills 検索の実装

Claude Code v2.1.121 がリリース。MCP サーバーの自動ロード設定や /skills のフィルタリング機能、不要なプラグインを整理する prune コマンドが追加されました。

リリース日 2026-04-28
バージョン v2.1.121
ツール Claude Code

主な変更点

今回のアップデートでは、開発ワークフローの効率化と環境整理に焦点を当てた機能改善が行われました。

  • MCP alwaysLoad: 設定ファイルで指定した MCP サーバーを常に読み込むオプションが追加され、セッション開始時の手間が軽減されました。
  • /skills フィルタリング: スキル一覧を表示する際にフィルタリングが可能になり、多数のスキルから目的のものを素早く見つけられます。
  • claude plugin prune: 未使用または不要になったプラグインを一括で削除し、ディスク容量と起動速度を最適化するコマンドが導入されました。
  • Fullscreen Mode の修正: 全画面表示時に発生していた表示崩れや入力の問題が解消されました。

影響や使いどころ

特に多くの MCP サーバーやカスタムスキルを運用しているプロユーザーにとって、管理の手間が大幅に削減されます。また、`prune` コマンドによる環境のクリーンアップは、長期間ツールを使用している開発者にとって有用なメンテナンス機能となります。

まとめ

Claude Code は着実に管理機能と信頼性を向上させています。MCP 連携の強化により、外部ツールを駆使したより複雑な開発タスクへの適応力が高まりました。
