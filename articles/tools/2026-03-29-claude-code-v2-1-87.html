Claude Code v2.1.87: Cowork Dispatch 修正とセッション追跡強化

Cowork機能のバグ修正に加え、APIリクエストへのセッションID追加やメモリ最適化を実施。

リリース日 2026-03-29
バージョン v2.1.87
ツール Claude Code

主な変更点 (v2.1.86 - 2.1.87)

Cowork Dispatch の修正

v2.1.87 にて、Cowork Dispatch（別エージェント・別タスクへの連携派遣）においてメッセージが正しく配信されない不具合が修正されました。

APIリクエストへのセッションID追加

すべてのAPIリクエストで x-claude-Code-Session-Id ヘッダーが送信されるようになりました。これにより、プロキシ等が本文を解析することなくセッションごとのリクエスト集計を行えるようになります。

VCS 除外リストのアップデート

.jj (Jujutsu) や .sl (Sapling) といった次世代バージョン管理システムのメタデータディレクトリが除外リストに追加されました。Grep検索やファイル自動補完で不要なシステムファイルが除外され、パフォーマンスが向上します。

改善とバグ修正

  • メモリとパフォーマンス: セッションの表示や長いログでのスクロールパフォーマンスの改善、TypeScript の実装へ移行による負荷軽減。
  • セキュリティ・状態管理: 不要なディスク書き込みの削減、OAuth トークンのセキュリティ向上、マルチウィンドウの不具合修正。
  • プロンプトキャッシング: ツール説明から動的コンテンツを削除し、Bedrock, Vertex, Foundry ユーザー向けにプロンプトキャッシュのヒット率を向上。

まとめ

v2.1.86からv2.1.87への一連のアップデートは、主にツールの安定性、各種バグ・エッジケースの解消、および大規模環境でのパフォーマンス向上に焦点を当てています。Cowork などの高度な連携機能も着実に洗練されています。
