Release Notes

ChatGPT: Windows サンドボックスの隔離強化とデバイスコードログインへの対応

OS レベルのネットワーク制御による安全性の向上と、CLI などのヘッドレス環境での利便性が改善されました。

リリース日 2026-04-09
バージョン April 2026 Update
ツール ChatGPT

主な変更点

Windows サンドボックスの隔離強化

Windows デスクトップ版において、AI がコードを実行する際のサンドボックスに対する OS レベルの制限機能が導入されました。 ネットワーク出力を特定のプロキシや許可されたドメインに限定するルールを設定可能になり、コード実行を通じたデータ漏洩リスクを大幅に低減します。

デバイスコードフローによるログイン

CLI やサーバーサイドでの利用を想定し、ブラウザを直接起動できない環境でもログイン可能な「デバイスコードフロー」がサポートされました。 表示されたコードを別のブラウザで入力するだけで、ヘッドレス環境でも安全に認証を完了できます。

影響や使いどころ

企業のセキュリティポリシーにより厳格なネットワーク制御が求められる環境でも、ChatGPT のコード実行機能をより安全に利用できるようになります。 また、Codex CLI などのツール利用時において、SSH 経由のリモート環境などからのログインがスムーズになります。

まとめ

今回のアップデートは「堅牢性」と「利便性」の両面を強化するものです。 特に Windows 環境でのサンドボックス制限は、エンタープライズ利用における大きな障壁を一つ取り払うものとなります。
