GitHub Copilot: 全プランの従量課金制移行と新規登録制限の発表

GitHub Copilot が 2026 年 6 月 1 日より全プランで従量課金制（pay-as-you-go）へ移行することを発表。これに伴い、新規のサブスクリプション登録が一時的に制限されています。

発表日 2026-04-27
種別 Pricing Update
ツール GitHub Copilot

主な変更点

GitHub は、サービスの持続可能性とスケーラビリティを確保するため、Copilot の料金体系を抜本的に見直しました。

  • 従量課金制への完全移行: 2026 年 6 月 1 日より、月額固定料金から、使用した AI トークン量に応じた従量課金制（pay-as-you-go）に移行します。
  • 新規登録の一時停止: 課金体系の移行準備とサーバー負荷の最適化のため、Individual プランを含む新規サブスクリプションの受け付けが一時的に制限されています。
  • リソース管理の強化: Copilot によるコードレビュー機能などが GitHub Actions の実行時間としてカウントされるようになり、リソース消費がより明確化されます。

影響や使いどころ

既存ユーザーは 6 月以降、自身の利用量に応じたコスト管理が求められるようになります。一方で、ライトユーザーにとってはコストが下がる可能性もあります。新規で導入を検討していた個人・チームは、制限が解除されるまでの間、代替ツールの検討や既存アカウントの調整が必要です。

まとめ

GitHub Copilot のこの決定は、AI 開発ツール市場全体が「一律料金」から「利用実態に即した課金」へと転換する予兆かもしれません。ユーザーは今後、AI の「賢さ」だけでなく「効率的な使い方」も意識する必要があります。
