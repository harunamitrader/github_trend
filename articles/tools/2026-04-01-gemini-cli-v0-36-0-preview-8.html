Gemini CLI v0.36.0-preview.8: preview 系列の最新タグに更新

Google の公式 Releases で v0.36.0-preview.8 が公開され、preview.7 から最新版タグが進みました。

リリース日 2026-04-01
バージョン v0.36.0-preview.8
ツール Gemini CLI

主な変更点

Google の公式 GitHub Releases では、2026-04-01 付で v0.36.0-preview.8 が新しい pre-release として公開されました。これにより、前日の v0.36.0-preview.7 から preview 系列の最新タグが一段進んだ形です。

公開ページには v0.36.0-preview.7...v0.36.0-preview.8 の比較リンクが付いていますが、確認時点の GitHub 表示では詳細な箇条書きはレンダリングされていませんでした。そのため、今回のログでは「最新版 pre-release への更新」が確定情報になります。

見ておくポイント

Gemini CLI は 0.36 系 preview を短い間隔で刻んでおり、安定版より先に修正や調整が流れる運用です。preview チャンネルを追っている場合は、このタグ更新自体に追随する意味があります。

まとめ

v0.36.0-preview.8 は、少なくとも公式 release ベースでは preview.7 の後継として公開済みです。詳細ノートは未表示でしたが、監視対象としては最新版更新を反映しておくべきタイミングです。
