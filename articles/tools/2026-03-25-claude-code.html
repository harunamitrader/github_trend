v2.1.81 (2026-03-24) のハイライト

最新の v2.1.81 では、主に安定性の向上とマイナーなバグ修正が行われました。 前日の v2.1.80 で導入された新機能群をより確実なものにするためのパッチリリースとなっています。

2026年3月の主要なアップデート

  • v2.1.80: ステータスラインの強化と、MCP（Model Context Protocol）メッセージ送信用の `--channels` プレビュー導入。
  • v2.1.79: `--console` フラグによるログイン機能と、ターン実行時間の表示切り替え機能。
  • v2.1.77: Claude 4.6 モデルへの最適化。出力トークン制限が大幅に拡張され、より複雑なコード生成が可能に。

MCP エコシステムへの統合

Claude Code は単なる CLI ツールを超え、MCP を通じた外部ツールとの連携を強力に推進しています。 これにより、ターミナルから直接ブラウザを操作したり、データベースにクエリを投げたりする「自律型エージェント」としての側面が際立っています。

まとめ

Anthropic は、開発者が使い慣れたターミナルという環境において、最も強力で信頼できる AI パートナーを提供し続けています。 Claude 4.6 の能力を最大限に引き出す CLI として、今後の進化からも目が離せません。
