GitHub Copilot: Claude Opus 4.7 が一般提供に

Copilot Pro+、Business、Enterprise 向けに Claude Opus 4.7 が段階展開され、複数の Copilot 画面から選択可能になります。

リリース日 2026-04-16
機能 Claude Opus 4.7
ツール GitHub Copilot

主な変更点

GitHub Changelog では、2026-04-16 付で Claude Opus 4.7 の一般提供が案内されています。GitHub は、Opus 4.7 について複数ステップのタスク、長い推論、ツール依存のワークフローで改善があると説明しています。

  • 対象プラン: Copilot Pro+、Business、Enterprise ユーザー向けに展開されます。
  • 利用面: VS Code、Visual Studio、Copilot CLI、Copilot Cloud Agent、github.com、GitHub Mobile、JetBrains、Xcode、Eclipse の model picker から選択できます。
  • 管理者設定: Business / Enterprise では管理者が Copilot settings で Claude Opus 4.7 policy を有効化する必要があります。

影響や使いどころ

複雑なリファクタリング、長い調査、Cloud Agent による修正などでは、モデルの推論力とツール実行の安定性が効きます。Opus 4.7 はそうした高負荷なエージェント作業の選択肢として見ておく価値があります。

補足

GitHub はサービス信頼性を高めるため、今後数週間で Copilot Pro+ の model picker では Opus 4.7 が Opus 4.5 / 4.6 を置き換える予定だとしています。プロモーション価格として、4月30日までは 7.5 倍の premium request multiplier で提供されます。

まとめ

今回の更新は、Copilot を IDE、CLI、Cloud Agent、モバイルまで横断して使うユーザーにとって、重い作業用モデルの選択肢を広げるものです。
