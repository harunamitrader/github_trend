Grok CLI v1.1.5: Grep WASM と Grok STT を統合

高速なコード検索とネイティブな音声認識を備え、自律エージェントとしての完成度が一段と向上しました。

リリース日 2026-04-18
バージョン v1.1.5
ツール Grok CLI

主な変更点

Grok を基盤とした自律型エージェント CLI「Grok CLI」の最新安定版 v1.1.5 がリリースされました。今回のアップデートでは、外部依存の削減と、開発中のコンテキストを阻害しないための新機能が目玉となっています。

  • Grok STT (Speech-to-Text) の導入: これまで音声処理に使用していた whisper.cpp を廃止し、xAI の Grok 自身の音声認識能力を統合しました。これにより、外部バイナリの依存が減り、認識精度と連携速度が改善されています。
  • 高速 Grep ツールの統合: ripgrep の WASM 版を組み込んだ専用検索ツールが追加されました。数万ファイル規模のプロジェクトでも、エージェントが瞬時にコードベースを走査・理解することが可能になります。
  • /btw コマンドの実装: 自律的なタスク（Plan Mode など）を実行している最中に、そのコンテキストを壊すことなく「ついでに聞きたいこと（By the way）」を質問できるようになりました。
  • MCP (Model Context Protocol) の改善: TUI から MCP の標準エラー出力を分離し、ターミナル表示の乱れを解消しました。

影響や使いどころ

v1.1.5 は、特に大規模なリポジトリで作業する開発者にとって強力な武器となります。Ripgrep の統合により「コードのどこに何があるか」の理解が爆速化し、エージェントが誤った推論をする確率が大幅に低下しています。また、音声による指示出しを使っているユーザーにとっても、よりスムーズな体験が提供されます。

まとめ

プレリリース版でのフィードバックを経て、v1.1.5 は非常に安定したバージョンとなっています。自律的なコーディング支援をコマンドラインから最大限に引き出すための、リファレンス的なツールへと進化を遂げています。
