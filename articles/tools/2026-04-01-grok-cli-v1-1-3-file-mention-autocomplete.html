Release Notes

Grok CLI v1.1.3: @ メンションのファイル補完を追加

@ 入力時のファイル autocomplete が加わり、対象ファイルを会話へ渡す操作が軽くなりました。

リリース日 2026-04-01
バージョン v1.1.3
ツール Grok CLI

主な変更点

superagent-ai の公式 GitHub Releases では、grok-dev@1.1.3 が 2026-04-01 18:51 UTC に公開されています。今回の変更点は 1 行で、@ メンション時のファイル autocomplete 追加です。

影響や使いどころ

大規模な機能拡張ではありませんが、会話中に特定ファイルを素早く参照コンテキストへ入れたいときの摩擦を減らせます。直前の 1.1.1 で verify workflow と headless batch mode、1.1.2 で computer sub-agent 切り替えが入っており、その流れの上で日常操作を磨く小さな更新と見てよさそうです。

まとめ

v1.1.3 は単機能の小粒なリリースですが、CLI ベースの対話効率を上げる変更です。Grok CLI を継続運用しているなら押さえておいてよい更新です。
