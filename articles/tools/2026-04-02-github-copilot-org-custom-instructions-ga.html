Release Notes

GitHub Copilot: Organization custom instructions が一般提供開始

Copilot Business / Enterprise 管理者が、組織全体の既定指示を github.com や code review、cloud agent に横断適用できるようになりました。

リリース日 2026-04-02
機能 Organization custom instructions
ツール GitHub Copilot

主な変更点

GitHub の公式 Changelog では、Organization custom instructions が 2026-04-02 付で generally available になったと告知されました。これは 2025 年 4 月に導入された機能の GA 化です。

管理者が設定した既定指示は、Copilot Chat on github.com、Copilot code review、Copilot cloud agent にまたがって適用されます。組織単位で応答方針や制約を揃えたい場合に効く更新です。

影響や使いどころ

Business / Enterprise 組織では、リポジトリごとの指示に加えて、全社的なコーディング方針やレビュー観点を既定値として注入しやすくなります。cloud agent まで適用範囲に含まれる点が重要です。

まとめ

この更新はモデル追加より地味ですが、Copilot をチーム標準化するうえでは実用度が高い変更です。管理者運用をしているなら確認優先度は高めです。
