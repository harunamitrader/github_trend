Release Notes

Claude Code v2.1.81: 自動化と遠隔操作のエクスペリエンス向上

`--bare` フラグによるスクリプト実行の最適化と、モバイル経由の承認フロー導入。

リリース日 2026-03-21
バージョン v2.1.81
ツール Claude Code

主な新機能

Claude Code v2.1.81 では、CI/CD や自動化スクリプトでの利用を想定した重要な機能が追加されました。

`--bare` フラグの導入

スクリプトから `-p` (print) モードで Claude Code を呼び出す際、UI フックや LSP (Language Server Protocol)、プラグインの同期、及びスキルディレクトリの走査をスキップできるようになりました。 これにより、オーバーヘッドを最小限に抑えた高速な実行が可能になります。

モバイル承認（Permission Relay）

許可機能を宣言したチャンネルサーバーを介して、PC 上でのツール実行承認プロンプトをスマートフォンに転送できるようになりました。 外出先からでもエージェントの作業を見守り、承認を与えることが可能です。

改善と修正

  • マルチセッションの安定化: 複数の Claude Code セッションを同時に実行している際、一方の OAuth トークンが更新されても他方での再認証が不要に。
  • 音声モードの修正: ネットワークエラー時に誤ったメッセージが表示される問題を解決。
  • MCPツール出力: モデルからのツール出力の視認性とバッシュモードの発見性を向上。

まとめ

今回のアップデートは、Claude Code を単なる対話ツールから、より堅牢な自動化基盤へと進化させる内容です。 特に `--bare` モードは、大規模なソースコード解析を自動化したい開発者にとって非常に強力な武器となるでしょう。
