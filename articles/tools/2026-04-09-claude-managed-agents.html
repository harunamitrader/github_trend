Release Notes

Claude Platform: Managed Agents 発表と ant CLI のパブリックベータ開始

自律型エージェントを安全に実行するフルマネージド基盤と、開発を加速する ant CLI が公開されました。

リリース日 2026-04-09
種別 Public Beta
ツール Claude

主な変更点

Managed Agents (Public Beta)

Anthropic サーバー上の安全なサンドボックス環境で AI エージェントを実行できるフルマネージド基盤「Managed Agents」がパブリックベータとして公開されました。 開発者は複雑なインフラを構築することなく、API 経由でエージェントに高度なタスクを実行させ、その結果を安全に取得できるようになります。

ant CLI の導入

API 管理やプロジェクト設定を容易にする新しいコマンドラインツール「ant CLI」が登場しました。 YAML 形式でのリソース定義、プロンプトのバージョニング、そして Claude Code とのネイティブな統合をサポートしており、開発効率を劇的に向上させます。

影響や使いどころ

これまで自律型エージェントの運用には安全な隔離環境の構築が不可欠でしたが、Managed Agents によりそのハードルが大きく下がります。 セキュリティが重視されるエンタープライズ環境において、AI エージェントの実装がより現実的になります。

まとめ

Anthropic は「モデル単体」の提供から、エージェントを実行する「プラットフォーム」への進化を鮮明にしました。 Managed Agents と ant CLI は、次世代の AI 開発における標準的なツールキットとなるでしょう。
