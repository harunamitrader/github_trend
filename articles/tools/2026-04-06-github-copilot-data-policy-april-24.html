Policy Update

GitHub Copilot: 2026-04-24 よりデータをモデル学習に使用へ（オプトアウト可）

Free / Pro / Pro+ ユーザーの操作データが AI モデル改善に使われるようになります。設定から随時オプトアウトできます。

発効日 2026-04-24
種別 ポリシー変更
ツール GitHub Copilot

変更の概要

2026-04-24 より、GitHub は Copilot Free・Pro・Pro+ ユーザーの操作データ（入力・出力・コードスニペット・関連コンテキストを含む）を、 AI モデルの改善・学習に使用するようになります。デフォルトはオプトインの扱いです。

Copilot Business・Enterprise ユーザーは対象外です。組織管理下のアカウントのデータは使用されません。

オプトアウトの手順

以下の手順でいつでも無効化できます。設定変更は即時反映されます。

  1. GitHub アカウント設定を開く
  2. Copilot → Privacy と進む
  3. 「Allow GitHub to use my data for AI model training」をオフにする

以前に類似の設定でオプトアウト済みの場合、その設定は引き継がれます。

注意点

対象はモデル改善のための学習用途です。コードの公開や第三者への共有とは別の話です。 それでも、プライベートなリポジトリや業務コードを扱う場合は、設定を確認しておくことを推奨します。

まとめ

個人プランで Copilot を使っており、データ学習への同意を避けたい場合は 2026-04-24 までに設定を確認しておくとよいでしょう。 Business・Enterprise プランのユーザーは今回の変更の影響を受けません。
