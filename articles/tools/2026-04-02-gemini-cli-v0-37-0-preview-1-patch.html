Gemini CLI v0.37.0-preview.1: preview.0 向けパッチを反映

最新 preview タグとして v0.37.0-preview.1 が公開され、preview.0 系列へパッチが即時反映されました。

リリース日 2026-04-02
バージョン v0.37.0-preview.1
ツール Gemini CLI

主な変更点

Google の公式 GitHub Releases では、v0.37.0-preview.1 が 2026-04-02 21:09 UTC に公開されました。今回の説明は短く、v0.37.0-preview.0 向けブランチへ特定コミットを cherry-pick して patch version を切り直したことが中心です。

つまり大型機能の追加ではなく、preview 系列をすぐ使いたい人向けの修正版タグです。nightly を追っているユーザーより、preview を基準に試すユーザーに向いた更新と言えます。

影響や使いどころ

前の nightly では subagent 設定反映や startup cleanup 並列化などが入っていましたが、今回の preview.1 は「新機能の積み増し」より「preview.0 の修正版提供」が主題です。preview 系列を固定参照している CI や検証環境では、こちらへ揃える意味があります。

まとめ

v0.37.0-preview.1 は小さい更新ですが、最新 preview タグを追うなら無視しない方がよい修正版です。preview.0 を採用する前提なら、そのまま preview.1 に寄せるのが自然です。
