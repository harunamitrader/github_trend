Release Notes

Gemini CLI v0.36.0: Task Tracker プロトコルの統合と A2A 強化

進捗管理の精度向上、マルチレジストリ対応、VSCode 連携の強化など、開発エージェントとしての基盤が大幅に進化。

リリース日 2026-03-31
バージョン v0.36.0
ツール Gemini CLI

主な変更点

Google の公式 CLI エージェントである Gemini CLI の最新安定版 v0.36.0（一部プレビュー版を含む一連の統合アップデート）がリリースされました。今回の更新では、単一のエージェントとしての能力向上に留まらず、マルチエージェント環境（A2A）や外部ツールとの連携強化に重点が置かれています。

1. Task Tracker プロトコルの統合

コアシステムのプロンプトに「Task Tracker プロトコル」が標準統合されました。これにより、進行中のタスクの進捗状況を AI がより正確に把握・管理できるようになり、複雑なマルチステップの作業における完遂率が向上しています。また、タスクの状態として新たに「blocked（中断中）」が追加されました。

2. Agent-to-Agent (A2A) とマルチレジストリ

エージェント間での承認コマンドやレジストリ探索機能が強化されました。複数のツールレジストリを同時にサポートするアーキテクチャが導入され、サブエージェントが使用できるツールを動的にフィルタリングすることが可能になっています。

3. 開発環境（VSCode）との連携強化

VSCode ワークスペース内の全てのフォルダが自動的に Gemini のコンテキストに追加されるようになりました。ディレクトリを跨いだ大規模なリファクタリングやコード探索が、手動の指示なしで開始できるようになっています。

その他の改善点

  • 動的なモデル解像度: ネットワーク状況やタスクの複雑さに応じてモデルの解像度を動的に調整する機能をサポート。
  • 自動更新の修正: スタンドアロンバイナリにおける自動アップデート処理の安定性が向上。
  • セキュリティ・ポリシーの緩和: 限定的なセッションにおけるブラウザツール使用制限を緩和し、利便性を向上。

まとめ

v0.36.0 は、Gemini CLI がシンプルなチャットボットから、プロジェクト全体の文脈を理解し、他のエージェントやツールと自律的に協調する「AI 同僚」へと進化するための重要なステップです。特に VSCode を活用する開発者にとって、コンテキスト管理の自動化は大きな恩恵となるでしょう。
