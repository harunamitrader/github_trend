Gemini CLI v0.41.0-preview.0: プレビュー版の刷新と安定性の向上

Gemini CLI v0.41.0-preview.0 が公開。Nightly で検証された ContextManager の安定化機能が統合され、プレビュー版としての信頼性が向上しました。

リリース日 2026-04-29
バージョン v0.41.0-preview.0
ツール Gemini CLI

主な変更点

週次のプレビュー版としてリリースされた v0.41.0-preview.0 では、直近のナイトリービルドで磨き上げられた機能が正式にプレビューへ統合されました。

  • ContextManager の安定化: メモリ消費を抑制しつつ、長時間セッションでの応答速度を維持する最適化が施されました。
  • セッション耐性の向上: 不意の接続断やエラー発生時の `/resume` 処理がより堅牢になり、作業の継続性が強化されました。
  • 内部依存関係の更新: セキュリティとパフォーマンスの観点から、コアライブラリのバージョンが最新化されています。

影響や使いどころ

ナイトリー版の実験的な機能を避けつつ、安定版よりも早く新機能を試したいユーザーにとって最適なバージョンです。大規模なコードベースを対象とした長時間のエージェント作業において、その安定性の恩恵を大きく受けることができます。

まとめ

Gemini CLI の開発サイクルは順調に推移しており、ナイトリーからのフィードバックが確実にプレビュー版の品質向上に繋がっています。
