Tool Update Log #408

AI Others Tools 統合アップデート: OpenClaw v2026.4.29, Grok 4.3, Claude Mythos Preview

OpenClaw v2026.4.29 のステアリング機能、Grok 4.3 の 100万トークン対応と価格改定、Claude Mythos の限定プレビュー開始など、2026年5月初旬の主要 AI 汎用ツール動向。

リリース日 2026-05-03
カテゴリ Others
ツール AI Others Tools

主な変更点

5月初頭の汎用 AI ツールは、自律型エージェントの操作性向上と、基盤モデルのさらなる高機能化が主なトピックとなっています。

  • OpenClaw v2026.4.29: エージェントの実行中に指示を追加・修正できる「Active Run Steering」機能を導入。Slack や Teams 等のメッセージングプラットフォームでの接続安定性も大幅に改善されました。
  • Grok 4.3: コンテキストウィンドウを 100万トークンへ拡張し、エージェント向けワークロードの価格を大幅に引き下げ。ビデオ入力やドキュメント生成機能もネイティブサポートされました。
  • Claude Mythos (Restricted Preview): Anthropic の次世代フラッグシップモデル「Mythos」が限定的なパートナー向けに公開。推論能力と創造性の飛躍的な向上が謳われています。
  • ChatGPT (GPT-5.5-Cyber): セキュリティプロフェッショナル向けの特化型モデル「GPT-5.5-Cyber」が、特定のプログラム参加者向けにロールアウトを開始しました。

影響や使いどころ

OpenClaw のステアリング機能により、エージェントが意図しない方向に進み始めた際、タスクを中断せずに軌道修正が可能になりました。Grok 4.3 の価格改定は、大量のデータ処理や継続的なエージェント運用を検討しているユーザーにとって強力な選択肢となります。

まとめ

モデルの性能競争だけでなく、エージェントとしての「扱いやすさ（ステアリング）」や「コストパフォーマンス」への最適化が鮮明になっています。
