Grok CLI v1.1.4-rc5: install script の tmp_dir エラーを修正

最新の RC では install script の tmp_dir 未定義エラーが修正され、導入時の失敗要因がひとつ潰されました。

リリース日 2026-04-03
バージョン v1.1.4-rc5
ツール Grok CLI

主な変更点

superagent-ai の公式 GitHub Releases では、grok-dev@1.1.4-rc5 が 2026-04-03 10:07 UTC に公開されています。変更点は 1 件で、install script にあった tmp_dir の未定義エラー修正です。

影響や使いどころ

RC 系列では 1.1.4-rc1 で verify command や binary release workflow、self-management CLI commands が入り、その後 rc2 から rc5 まで導入まわりの細かい整備が続いています。今回の修正は派手ではありませんが、セットアップ失敗を減らす意味で重要です。

まとめ

v1.1.4-rc5 は導入安定性のための小規模パッチです。Grok CLI の新しい配布導線を試しているなら、この RC に追従しておく価値があります。
