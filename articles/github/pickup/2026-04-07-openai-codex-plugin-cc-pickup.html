codex-plugin-cc

Claude CodeからOpenAIモデルを直接操り、コードレビューや重い実装を「委譲」できる強力なプラグイン。

Published 2026-04-07
Repository openai/codex-plugin-cc

概要

Claude Code内からOpenAIのモデル（Codex/GPT-4o）を呼び出し、コードレビューやタスクの委譲を行うためのプラグインです。Claudeのインターフェースを離れることなく、OpenAIの知見、特にコード解析能力を直接開発フローに取り込めます。

注目の機能

  • 専用コマンドの追加: /codex:review（通常レビュー）や /codex:adversarial-review（さらに厳格・批判的な視点でのレビュー）などのコマンドが利用可能になります。
  • バックグラウンドタスク委譲: /codex:rescueコマンドを通じて、大きな改修タスクや修正作業をOpenAI側に任せ、Claude側での作業を止めずに並列して処理させることができます。
  • ハイブリッド開発の実現: 特定のタスクはOpenAIに、主要な設計や対話はClaudeに、といった「モデルの適材適所」による開発がシームレスに行えます。

活用シーン

Claudeが得意とするコンテキスト理解を活かしつつ、OpenAI（Codex）による厳格な静的解析や重厚なコード改修の力を借りたい場合。モデルの「複数視点」を得ることで、より品質の高い成果物を目指せます。

ClaudeとOpenAIという二大巨頭の知性を1つのCLIターミナルで統合する、まさに「最強」を目指すためのツールです。
