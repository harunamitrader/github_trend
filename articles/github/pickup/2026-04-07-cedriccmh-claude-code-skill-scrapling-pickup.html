claude-code-skill-scrapling

Claude Codeに「Cloudflareも突破可能な高度なWebスクレイピング能力」を付与する特化型スキル。

Published 2026-04-07
Repository Cedriccmh/scrapling

概要

Claude Codeの「スキル」として動作する、強力なWebスクレイピングライブラリ「scrapling」の拡張機能です。Claudeがウェブサイトの内容をより正確かつ効率的に読み取り、データを抽出することを可能にします。

主な特徴

  • 最適なフェッチャーの自動選択: サイトの構造や保護状態に応じて、最適なスクレイピング手法（StealthyFetcher, DynamicFetcherなど）を自動で選定します。
  • Cloudflareバイパス機能: 通常のスクレイパーでは難航するCloudflareやWAFなどのボット対策を回避する機能を備えています。
  • Python連携: Claude CodeがPythonスクリプトを生成・実行する形で動作し、セッション維持やログインが必要なページにも対応可能です。

活用シーン

Claude Codeを使って開発している際、「外部ドキュメントを最新の状態で読み込みたい」「動的なコンテンツが含まれるAPIドキュメントを解析したい」といった場面で、標準の読み込み機能では限界がある場合に威力を発揮します。

Claude Codeのエージェント性能を「外部情報の取得」という面で大幅に底上げする、実用性の高い拡張スキルです。
