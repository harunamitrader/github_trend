これは何か

世界中のニュース、地政学的な出来事、金融データ、インフラ（航空機・船舶等）の状況を一つの画面に集約し、AIが要約・分析を行う「状況把握（Situational Awareness）」のためのリアルタイム・ダッシュボードです。

何ができるか

500以上のニュースフィードの監視、地政学・災害・経済シグナルの相関分析、3D地球儀と2Dマップによる45層のデータ可視化、92の取引所をカバーする金融レーダー、そしてローカルAI（Ollama等）によるインテリジェンス要約が可能です。

目立つポイント

非常に高度なデータ可視化技術（globe.gl / deck.gl）と、ローカル環境で動作するAIを組み合わせている点です。プライバシーを重視しつつ、膨大なオープンソース・インテリジェンス（OSINT）を個人レベルで処理できるように設計されています。

セットアップや使い方の流れ

git clone して npm install 後、npm run dev で起動します。特別なAPIキーなしでも基本機能が動作し、Tauriベースのデスクトップアプリ（.exe / .dmg）としてインストールして使うこともできます。

どんな人向けか

地政学アナリスト、投資家、ジャーナリスト、あるいは世界の動向を常に鳥瞰的に把握しておきたい、テック・インテリジェンスに関心の高いユーザーに最適です。

注意点

扱うデータ量が膨大であるため、ブラウザやマシンのリソースを一定以上消費します。また、高度なAI要約機能を利用するには Ollama 等のセットアップが推奨されます。

まとめ

「パランティア（Palantir）」のような高度な状況把握システムをオープンソースで実現しようとする野心的なプロジェクトです。世界で何が起きているかを「データ」で可視化する強力なツールと言えます。
