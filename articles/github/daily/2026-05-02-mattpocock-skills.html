これは何か

AIコーディングツール（特に Claude Code）向けに最適化された、自律的な開発、リファクタリング、アーキテクチャ設計を支援するプロフェッショナルなスキル定義集です。

何ができるか

エージェントに対して「TDDモードへの移行」「アーキテクチャの変更提案」「依存関係のクリーンアップ」などの高度な指示を、構造化されたコマンドとして実行させることができます。

目立つポイント

現場のエンジニアが日々直面する課題に基づいて設計されており、エージェントが文脈を正しく理解し、人間と高い同期レベルで協調作業を行えるようになります。

セットアップや使い方の流れ

プロジェクトの `.claude` フォルダやグローバル設定にコピーするだけで、標準的なエージェント機能が劇的に強化されます。

どんな人向けか

AIとのペアプログラミングにおいて、より高度な抽象化や品質管理を求めているプロフェッショナル・エンジニア。

注意点

スキル定義が特定のツール（Claude Code等）の仕様に依存している場合があるため、使用環境に合わせて適宜調整が必要な場合があります。

まとめ

「AIに丸投げ」するのではなく、「AIを最高のアシスタントにする」ための、具体的かつ実戦的な知恵の集合体です。
