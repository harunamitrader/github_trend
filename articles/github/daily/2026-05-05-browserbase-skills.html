これは何か

browserbase/skills は、Anthropic の Claude (Claude Code 等) を含む AI エージェントに、高度な Web ブラウジングおよび操作能力を付与するための SDK です。ヘッドレスブラウザのホスティングサービスである Browserbase をバックエンドに使用しています。

注目のポイント

  • MCP 対応: Model Context Protocol に準拠しており、エージェントから「スキル」として容易に呼び出すことができます。
  • 高度なインタラクション: 単なるテキスト取得だけでなく、クリック、入力、スクロールといった人間のようなブラウザ操作をエージェントに代行させることが可能です。
  • 認証・セッション管理: 複雑なログインが必要なサイトや、動的なコンテンツの読み込みもBrowserbase のインフラ上で安定して処理します。

筆者の視点

エージェントが自律的に調査を行い、フォームを送信し、データを収集するための「手」となるライブラリです。特に Claude Code との相性が良く、開発者がターミナルから出ることなく Web 上のドキュメントをエージェントに読ませる、といったワークフローが非常にスムーズになります。
