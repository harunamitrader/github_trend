これは何か

Google や Amazon といったトップテック企業の面接を突破するために作成された、コンピュータサイエンス（CS）の独学用学習プランです。筆者の John Washam 氏が、数千時間に及ぶ学習を経て実際にソフトウェアエンジニアとしてのキャリアを築いた経験をもとにまとめられています。

何ができるか

体系的なカリキュラムに沿って学習することで、学位を持たない状態からでもコンピュータサイエンスの核心的な知識を身につけることができます。データ構造、アルゴリズム、システム設計、数学、さらには行動面接の対策まで、エンジニアとして必要な要素がすべて網羅されています。

目立つポイント

単なるリンク集ではなく、各トピックに対して学習すべき優先順位や動画リソース、練習問題が整理されています。世界中の開発者から絶大な支持を受けており、34万件以上のスターがその価値を証明しています。

セットアップや使い方の流れ

GitHub リポジトリにある README を読み、自分の現在の知識レベルと目標とする企業に合わせて学習項目を選択します。チェックリスト形式になっているため、進捗を管理しながら自分のペースで進めることができます。

どんな人向けか

非 CS 学位からエンジニアを目指す方、基礎知識を再整理したい現役開発者、そしてトップ企業への転職を考えている方に最適です。内容は非常に膨大ですが、本気でエンジニアとしての基礎体力をつけたい人にとっての「聖書」と言える存在です。

注意点

すべてを完了するには数ヶ月から 1 年以上の膨大な時間が必要です。挫折を防ぐためには、一度にすべてをやろうとせず、小さな目標を積み重ねることが推奨されます。

まとめ

「エンジニアになるための最短距離」ではなく「最も確実な王道」を提示するガイドです。これを完遂したとき、あなたは世界中のどこでも通用するソフトウェアエンジニアの基礎を手にしているはずです。
