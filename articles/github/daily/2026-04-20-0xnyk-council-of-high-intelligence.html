これは何か

council-of-high-intelligence は、歴史的な哲学者、科学者、そして現代のテックリーダーなど18名の人格（AI Persona）からなる「最高知能評議会」を召集し、あなたの複雑な意思決定を審議させるためのツールです。 単一の AI に答えを求めるのではなく、複数の LLM プロバイダーを横断して異なる知性を戦わせることで、バイアスのない深い洞察を得ることを目的としています。

何ができるか

  • 多段階審議プロトコル: 独立分析 → 相互反対尋問 → 最終結論という3段階のプロセスを経て議論を深めます。
  • 多様なプロバイダーの活用: 反応の均質化を防ぐため、評議会員を Claude, OpenAI, Gemini, Ollama(ローカル) など異なるプロバイダーへ自動的に振り分けます。
  • 18名の評議会員: アリストテレス、ファインマン、カーネマン、リーナス・トーバルズ、宮本武蔵、アンドレイ・カーパシーなど、古今東西の知性が揃っています。

目立つポイント

  • 対立構造の強制（Polarity Pairs）: 「ソクラテス vs ファインマン（前提の破壊 vs 第一原理からの再構築）」や「サン・ズ（孫子） vs アウレリウス（外的な勝利 vs 内的な規律）」など、あえて対立的な視点を持つペアで議論を戦わせます。
  • バイアス・エコーチェンジの抑制: 意見の一致が早すぎる場合に強制的に反対意見（スチールマニング）を生成させる「ディセント・クォータ（反対意見ノルマ）」機能を搭載しています。
  • 課題のリフレーミング: 各メンバーは分析の前にまず「質問そのもの」を自分なりに定義し直します。これにより、そもそも問いが間違っていないかを検証します。

セットアップや使い方の流れ

  1. インストール: リポジトリをクローンし、./install.sh を実行して Claude Code や Codex にインストールします。
  2. 実行: /council [あなたの悩みや決断すべきこと] を入力します。
  3. モード選択: 徹底審議のフルモードのほか、簡単な判断のための --quick、2名での対立軸を探る --duo モードが選べます。
  4. 審議の確認: 異なる知性たちが互いの矛盾を突き合いながら結論を導き出していく過程が表示されます。

どんな人向けか

  • 重要なビジネス戦略、技術選定、あるいは倫理的なジレンマに直面しているリーダー。
  • 単一の AI の「尤もらしい回答」ではなく、あえて「批判的な視点」や「多様な代替案」を求めている人。
  • エージェント・スウォーム（大規模なエージェント群）による意思決定の自動化に興味があるエンジニア。

注意点

  • 全プロバイダーの環境: 全ポテンシャルを引き出すには、Claude, OpenAI, Gemini 等の API キーが設定されていることが望ましいです（もちろん単一プロバイダーでも動作します）。
  • 審議コスト: 18名の AI が複数回対話するため、トークン消費量は単一プロンプトよりも多くなります。

まとめ

「最高知能評議会」は、AI を単なる『回答者』から、自らの思考を磨き上げるための『壁打ち相手としての評議会』へとアップグレードさせるツールです。 歴史上の賢人たちが現代の技術スタックやビジネス課題について激論を交わす様子は、エンジニアリング的にも知的エンターテインメントとしても非常に興味深い試みと言えます。
