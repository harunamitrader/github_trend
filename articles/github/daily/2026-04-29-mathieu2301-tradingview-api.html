TradingView-API

TradingViewのリアルタイム価格やインジケーター値をPython/Node.jsから取得・操作できる非公式API。

これは何か

TradingViewのウェブサイトからリアルタイムの市場価格やテクニカル指標のデータを取得するためのライブラリ。非公式ながら、通常のブラウザアクセスをエミュレートすることで、API制限を超えた高度なデータ取得を可能にしています。Node.js環境を中心に、Python等のラッパーも存在します。

何ができるか

リアルタイムの株価・暗号資産価格の取得はもちろん、RSIやMACDといったテクニカル指標の値を取得できます。また、ストラテジーのバックテスト実行やチャート上の描画オブジェクトの取得、さらには「招待制インジケーター」の値にもアクセス可能です。

目立つポイント

最大の強みはWebSocketサーバーのエミュレーションによるリアルタイム性です。ブラウザ版と遜色ない速度でデータを更新でき、スクレイピングの枠を超えた「TradingViewのバックエンド化」を実現します。リプレイモードのシミュレーションや、公式では制限されている無料プランでの複数時間軸データ取得などもサポートされています。

セットアップや使い方の流れ

基本的には npm i @mathieuc/tradingview でインストール可能です。WebSocketクライアントを初期化し、対象のシンボル（例: BINANCE:BTCUSDT）を指定して接続します。onUpdate 等のイベントを介して、最新の価格や指標の更新データをリアルタイムで受け取る実装になります。

どんな人向けか

自作のトレーディングBotを開発しているエンジニアや、独自のバックテスト環境を構築したい個人投資家に最適です。特に、TradingViewの使いやすいUIで分析した結果を、そのままプログラム側の「トリガー」として利用したい場合に真価を発揮します。

注意点

非公式ライブラリであるため、TradingView側のフロントエンドの仕様変更により突然動作しなくなるリスクがあります。また、利用規約（ToS）に抵触する可能性があるため、過度なリクエストや商用利用には細心の注意が必要です。

まとめ

TradingViewの膨大な市場データと強力な分析エンジンを、外部プログラムから直接制御可能にする強力なAPI。自律型トレードエージェントの「感覚器官」として、あるいはデータ分析のパイプラインとして、非常に高い実用性を誇るリポジトリです。
