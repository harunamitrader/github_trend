これは何か

opennews-mcp は、Claude や Cursor などの AI エージェントから最新ニュースへ即座にアクセスできるようにする MCP (Model Context Protocol) サーバーです。 特に仮想通貨（Crypto）や金融領域に強く、世界中の 84 以上のニュースソース、オンチェーンデータ、取引所のリスティング情報、ミームコインのトレンドなどを一元的に集約し、AI による分析を加えて提供します。

何ができるか

  • リアルタイム・ニュース集約: Bloomberg, CoinDesk, 取引所のお知らせなど、多様なソースから最新情報を取得。
  • AI による自動分析: 各記事に対して重要度（0-100スコア）、売買シグナル（Long/Short/Neutral）、多言語要約を自動付与。
  • 高度な検索・フィルタリング: 特定の通貨（BTC, ETH等）、データ種別（Engine Category）、スコアの高さなどに基づいてニュースを絞り込み。
  • WebSocket による購読: 最新のアップデートをリアルタイムで受け取ることが可能です。

目立つポイント

  • 金融・投資特化: 単なるニュースリーダーではなく、「Trading Signals」や「Impact Score」といった、投資判断に直結するメタデータが充実しています。
  • MCP サーバーとしての使いやすさ: Claude Desktop や Cursor に一行のコマンド、あるいは設定ファイルへの追記だけで統合でき、エンジニアのワークフローを邪魔しません。
  • 圧倒的な網羅性: ニュース（News）、オンチェーン（On-chain）、上場情報（Listing）、ミーム（Meme）、マーケット（Market）、予測市場（Prediction）の6つのエンジンカテゴリーを網羅しています。

セットアップや使い方の流れ

  1. Token取得: https://6551.io/mcp で API トークンを取得します。
  2. インストール: claude mcp add opennews または設定ファイル（claude_desktop_config.json 等）に、取得したトークンと実行パスを登録します。
  3. 実行（問いかけ）: AI に対して「最新の BTC 関連ニュースを教えて」「今のマーケットで重要度の高い情報を要約して」と指示するだけで、ツールが起動し情報を取得します。

どんな人向けか

  • 膨大なニュースの中から、AI を使ってノイズを除去し、重要な情報だけを効率的にキャッチアップしたい投資家。
  • 特定の通貨やトレンドに関するオンチェーン情報を、ブラウザを開かずに IDE や端末内だけで完結させたいトレーダー。
  • 自身の AI エージェント（Claude Code 等）に、最新の市場データやニュース検索能力を付加したい開発者。

注意点

  • トークンの管理: API アクセスには 6551.io のトークンが必要なため、管理には注意してください。
  • 情報の取捨選択: AI によるレーティング（スコア）は便利ですが、最終的な投資判断は必ず一次情報を確認した上で行う必要があります。

まとめ

opennews-mcp は、AI エージェントを「ただのプログラム」から「敏腕の情報アナリスト」へと変貌させる強力なプラグインです。 84を数える膨大なソースを MCP という統一インターフェースで扱えるメリットは大きく、特に情報の断片化が激しい Web3・金融界隈において、強力な武器になるでしょう。
