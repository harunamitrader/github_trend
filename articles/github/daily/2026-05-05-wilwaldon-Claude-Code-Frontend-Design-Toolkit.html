これは何か

Anthropic の AI コーディングエージェント「Claude Code」が、より美しく使いやすいフロントエンドを出力できるようにするための、設定ファイル（CLAUDE.md）や推奨ツールのリポジトリです。

何ができるか

デザイン原則の適用、Lucide アイコンの適切な使用、Tailwind CSS の最適化、そしてコンポーネントライブラリの正しい導入方法などを Claude Code に指示するためのベストプラクティスを提供します。

目立つポイント

すぐにコピーして使える CLAUDE.md のスニペット、デザイン補助に特化した MCP サーバーの紹介、そして Claude Code の公式ドキュメントだけでは到達しにくい「審美的な出力」のためのプロンプトハックが満載です。

セットアップや使い方の流れ

README に記載されている設定をプロジェクトの `CLAUDE.md` に追記するか、推奨されている MCP サーバーをインストールすることで、次回のセッションから Claude の出力が洗練されたものに変わります。

どんな人向けか

Claude Code を使ってフロントエンドを構築しているが、生成される UI の見た目に満足していない開発者や、デザイン作業をより自動化したいエンジニアに最適です。

注意点

ツールの組み合わせによっては Claude の推論時間が長くなる可能性があるため、プロジェクトの規模や必要性に応じて設定を調整することが推奨されます。

まとめ

「コードが動く」のその先へ。AI エージェントを優秀なフロントエンドデザイナーへと変貌させるための決定版リソースです。
