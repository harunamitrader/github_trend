maigret

「ユーザー名ひとつで 3000 以上のサイトを徹底調査」。OSINT の強力な味方、デジタルフットプリントの可視化ツール。

記事作成日 2026-05-05
種別 GitHub Trending
Star数 24,676

これは何か

maigret は、特定のユーザー名がどのアカウント（SNS、フォーラム、Web サイト）で使用されているかを高速に特定する OSINT（オープンソース・インテリジェンス）ツールです。有名な Sherlock をベースにしつつ、対応サイト数やプロファイリング能力を大幅に拡張しています。

注目のポイント

  • 圧倒的な対応数: 3,000 以上のサイトを対象に、再帰的にユーザー情報を検索します。
  • 詳細なレポーティング: 単なる「存在確認」にとどまらず、プロフィール画像、リンクされたメールアドレス、名前、位置情報などを抽出してレポート化します。
  • 多機能な出力フォーマット: HTML, PDF, JSON, CSV など、調査目的に合わせた形式で結果を出力可能です。

筆者の視点

サイバーセキュリティの調査や、自分自身のデジタルフットプリント管理において非常に強力なツールです。情報の集約能力が非常に高いため、倫理的な使用が求められますが、OSINT ツールとしての完成度はトップクラスです。Docker での実行も容易で、すぐに調査を開始できる点も魅力です。
