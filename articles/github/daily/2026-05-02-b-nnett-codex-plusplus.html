これは何か

Codex++ は、OpenAI の Codex デスクトップアプリを「パッチ」して、カスタム機能やUI改善を後付けで追加できるようにする非公式の拡張機能（Tweak）システムです。本体を再ビルドすることなく、自分好みの機能（ショートカット、UI改善、追加のサイドバーなど）を注入できるのが最大の特徴です。

何ができるか

Codex 本体をパッチして専用のローダーを組み込むことで、外部の ESM モジュール（Tweak）を起動時に読み込ませることができます。アプリ内に「Tweaks」設定タブが追加され、インストール済みの拡張機能を個別に有効・無効化したり、設定を変更したりすることが可能です。また、自動修復機能（repair）を備えており、アプリのアップデートでパッチが外れても簡単に復元できます。

目立つポイント

1. **再ビルド不要**: アプリのバイナリを直接書き換えるのではなく、asar ファイルにフックを仕掛ける仕組みのため、拡張機能の開発が非常にスムーズです。 2. **管理 UI の統合**: 設定画面に「Tweaks」項目が統合され、ユーザーフレンドリーな操作が可能です。 3. **強力な CLI**: codexplusplus コマンドを使用して、ステータス確認、修復、アップデート、アンインストールなどが簡単に行えます。 4. **GitHub 連携**: 各 Tweak は GitHub リポジトリと紐付けることができ、更新チェックも自動化されています。

セットアップや使い方の流れ

Bun がインストールされている環境であれば、bun install -g github:b-nnett/codex-plusplus を実行後、codexplusplus install でセットアップが完了します。macOS/Linux/Windows それぞれに専用のインストールスクリプトも用意されています。インストール後は、アプリの設定画面から Tweak を管理できます。

どんな人向けか

Codex デスクトップアプリを愛用しており、さらに自分好みのショートカットやUIにカスタマイズしたいパワーユーザーや開発者。また、独自の拡張機能を JavaScript/TypeScript で自作してみたいハッカーに最適です。

注意点

非公式のプロジェクトであり、OpenAI の公式サポートはありません。アプリのアップデート内容によっては動作が不安定になる可能性や、パッチの再適用が必要になる場合があります。また、macOS ではアプリの署名が変更されるため、初回起動時に手動での許可が必要になることがあります。

まとめ

Codex++ は、デスクトップ版 Codex の体験を劇的に向上させるための「魔法のパッチ」です。エコシステムが整いつつあり、UI改善や生産性向上ツールとしてのポテンシャルが非常に高い、AIコーダー注目のプロジェクトです。
