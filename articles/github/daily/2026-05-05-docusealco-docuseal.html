これは何か

DocuSign のような電子署名・ドキュメント管理機能を、自身のサーバーに構築（セルフホスト）できるオープンソースプラットフォームです。

何ができるか

PDF などのドキュメントをアップロードし、署名が必要な箇所をドラッグ＆ドロップで設定、特定の相手に署名依頼を送付し、法的に有効なデジタル署名を収集できます。

目立つポイント

ブラウザ完結の直感的なエディタ、複数の署名者への順次依頼、モバイル対応の署名画面、PDF の自動生成、そして Docker による極めて容易なセットアップが特徴です。

セットアップや使い方の流れ

公式の Docker Compose ファイルを使用して `docker compose up -d` を実行するだけで、すぐに署名プラットフォームが立ち上がります。

どんな人向けか

高額なクラウド署名サービスのコストを削減したい企業や、個人情報を自社サーバー内で完結させたいプライバシー重視の組織に最適です。

注意点

法的な有効性は各国の電子署名法に依存するため、重要な契約に使用する際は現地の法規制との適合性を確認する必要があります。

まとめ

商用サービスに依存せず、セキュアで自由なドキュメント署名環境を構築できる、現代のビジネスに必須の OSS です。
