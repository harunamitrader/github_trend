これは何か

映画、音楽、写真などのパーソナルメディアを集中管理し、Web ブラウザ、モバイル、スマート TV などからストリーミング再生できる無料のメディアサーバーソフトです。

何ができるか

自宅の PC や NAS に保存された動画ファイルを、Netflix のような美しい UI で整理・表示し、外出先からもセキュアにアクセスして視聴することができます。

目立つポイント

完全にフリーで広告や追跡機能がなく、Plex や Emby といった他社製品と異なり全ての機能（ハードウェアエンコード等）が無料で利用できる点が最大の特徴です。

セットアップや使い方の流れ

Windows, Linux, macOS に加え、Docker での構築もサポートされています。インストール後、ブラウザからメディアフォルダを指定するだけでセットアップが完了します。

どんな人向けか

大量の録画や動画データを所有しており、サブスクリプション料金を払わずに高品質なメディアセンターを自前で運用したいユーザーに最適です。

注意点

サーバーのスペック（特に GPU）が動画の変換（トランスコード）性能に直結するため、多人数で同時視聴する場合は相応のハードウェアが必要です。

まとめ

プライバシーと自由を妥協したくない、究極のホームメディア環境を構築するためのデファクトスタンダードです。
