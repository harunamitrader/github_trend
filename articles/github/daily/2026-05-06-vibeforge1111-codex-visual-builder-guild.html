これは何か

Codex Visual Builder Guild は、Codex (Antigravity) や Spark Skill Graphs での UI 開発を劇的に向上させるための、16 種類の専門的な AI スキルを集約した「ギルド（専門集団）」です。AI がコードを書いて終わりにするのではなく、実際にアプリを動かし、スクリーンショットを撮り、Vision モデルで視覚的にレビューし、問題を各専門家に割り当てて修正するという「ビジュアル・ビルダー・ループ」を構築します。

何ができるか

・実機のスクリーンショットに基づいた、正確なビジュアル QA ループの実行。
・モバイル、デスクトップ、タブレットなど、マルチデバイス間でのレイアウト整合性チェック。
・imagegen を活用した、プレースホルダではない実用的な UI 資産（ヒーロー画像、アイコン等）の自動生成。
・アクセシビリティ、インタラクション（ホバー・フォーカス状態）、ブランドの一貫性など、特定の視点を持つ 16 人の「専門家」へのタスク委譲。
・修正前後の比較スクリーンショットと、再利用可能なデザインルールの自動抽出。

目立つポイント

単一のプロンプトで「綺麗にして」と頼むのではなく、16 種類の「レンズ（視点）」を定義している点がユニークです。例えば、レスポンシブ専門のレンズ、アクセシビリティ専門のレンズ、デザインシステム（トークン）専門のレンズなどが存在し、メインの監督エージェントが問題に応じて適切な専門家を召喚します。これにより、AI 特有の「なんとなくそれっぽい」デザインから脱却し、細部まで意図の通ったプロダクト品質を実現します。

セットアップや使い方の流れ

Codex Desktop を使用している場合、リポジトリをクローンして npm run install:codex を実行するだけで、ネイティブスキルとしてインストールされます。その後、Codex に対して「Use codex-visual-builder-guild to run the visual builder loop on this app.」と指示するだけで、ループが開始されます。Spark Skill Graphs ユーザー向けには、YAML 形式のスキルグラフ定義も提供されています。

どんな人向けか

・AI を使って爆速でプロトタイプを作りたいが、デザインの細部（余白、対比、モバイル対応など）に妥協したくない開発者。
・複数の AI エージェントを連携させて、より高度で自律的な開発ワークフローを構築したいパワーユーザー。
・Codex や Spark Skill Graphs の可能性を最大限に引き出したいコミュニティメンバーに最適です。

注意点

このギルドは、Codex が「実際にアプリを起動してスクリーンショットを撮影できる環境（ブラウザ操作ツールやローカルサーバー起動権限）」を持っているときに最大の威力を発揮します。また、Vision モデル（GPT-4o や Claude 3.5 Sonnet 等）の能力に依存するため、使用するモデルの視覚認識精度によって結果が左右される場合があります。

まとめ

Codex Visual Builder Guild は、AI によるソフトウェア開発を「コード生成」の段階から「プロダクト開発」の段階へと押し進める強力なツールセットです。人間がデザイナーとディレクターの間を行き来するように、AI にも視覚的なフィードバックと専門的な分業体制を与えることで、これまで手動で行っていた面倒な微調整や QA を自律的に解決できるようになります。
