これは何か

Anthropic の CLI ツール「Claude Code」をより効率的に利用するために、Obsidian（ナレッジ管理）と Graphify（コードグラフ生成）を組み合わせたメモリ構築システムです。

何ができるか

コードベース全体を知識グラフ化して Obsidian で管理し、Claude Code が必要な情報だけをピンポイントで参照できるようにすることで、セッションごとのトークン消費を劇的に削減します。

目立つポイント

Graphify による自動ドキュメント生成、Obsidian による永続的なメモリ保存、そして Karpathy 流のワークフローを参考にしたチャット履歴のインポートパイプラインが特徴です。

セットアップや使い方の流れ

Graphify でプロジェクトをスキャンし、生成されたノートを Obsidian Vault に保存、その後 Claude Code にそのフォルダを参照させることで、文脈を維持した開発が可能になります。

どんな人向けか

Claude Code の高額な API 利用料を抑えたいエンジニアや、大規模プロジェクトでコンテキスト制限に悩んでいる開発者に最適です。

注意点

セットアップには Graphify や Obsidian の基本知識が必要であり、ファイル名に特殊文字が含まれる場合のバッチ処理など、一部手動での調整が必要な場合があります。

まとめ

「ツールの組み合わせ」で AI の限界を突破する、コストパフォーマンスと生産性を両立させるための高度なハック集です。
