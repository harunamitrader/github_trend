これは何か

Windows 用のオープンソースな画面キャプチャ・スクリーン録画ツールです。単純なスクリーンショット撮影だけでなく、ファイル共有サイトへの自動アップロードや、多様な編集・加工機能を備えており、15 年以上にわたって開発が続けられている非常に成熟したユーティリティです。

何ができるか

全画面、ウィンドウ、特定領域のキャプチャはもちろん、GIF アニメーションや MP4 での動画録画が可能です。撮影後、画像を Imgur や Google Drive に自動アップロードし、共有リンクをクリップボードにコピーするといったワークフローをワンクリックで完遂できます。

目立つポイント

「ホットキー」のカスタマイズ性が極めて高く、自分好みのショートカットを無数に作成できます。また、OCR（画像内のテキスト抽出）、カラーピッカー、ハッシュ値計算、QR コード生成といった、開発者やクリエイターに便利な「おまけツール」も大量に内蔵されています。

セットアップや使い方の流れ

公式サイトまたは GitHub からインストーラーをダウンロードしてインストールします。起動後はトレイアイコンから設定を開き、キャプチャ後の動作（編集を開く、保存する、アップロードするなど）を自分のワークフローに合わせて構成するだけです。

どんな人向けか

頻繁にスクリーンショットを撮ってチームに共有するエンジニア、手順書を作成するライター、あるいは単純に Windows の標準機能では満足できないパワーユーザーに最適です。軽量かつ多機能で、広告も一切ありません。

注意点

機能が非常に多いため、初めて使うときは設定画面の複雑さに圧倒されるかもしれません。まずはデフォルト設定で使い始め、必要に応じてホットキーを一つずつ追加していくのがおすすめです。

まとめ

Windows 環境において、これ以上の多機能さと信頼性を両立したキャプチャツールは他にありません。一度使い始めると、他の OS に移ったときに最も恋しくなるツールの一つになるでしょう。
