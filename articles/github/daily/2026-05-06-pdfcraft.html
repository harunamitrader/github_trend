これは何か

PDFCraftは、Next.js 15とWebAssembly（WASM）を活用した、完全クライアントサイド動作のPDFツールキットです。90種類以上の専門ツールをブラウザ上で無料で利用でき、ファイルのアップロードを一切行わずに高度なPDF編集や変換が可能です。

何ができるか

PDFの結合・分割はもちろん、Office文書や画像、電子書籍（EPUB/MOBI）、Markdown、さらにはHTMLやURLからのPDF変換にも対応しています。また、PDFから画像やテキスト（JSON形式）、Word/Excelへの書き出し、パスワード設定、メタデータの削除など、実務で必要なあらゆる操作を網羅しています。

目立つポイント

最大の特長は「プライバシー」と「多機能性」の両立です。PyMuPDFのWASM版などを用いることで、従来はサーバーサイドで行っていた重い処理をブラウザ内で完結させています。Next.js 15やTailwind CSS 4といった最新のスタックで構築されており、UIも非常にモダンで使い勝手が良いのも魅力です。

セットアップや使い方の流れ

ウェブサイトにアクセスするだけで即座に利用可能です。ローカルで動かしたい場合は、GitHubからクローンしてnpm install & devで実行できるほか、Dockerイメージも提供されており、セルフホストも容易です。

どんな人向けか

機密性の高いドキュメントを扱うビジネスパーソン、複雑なPDF編集を頻繁に行うクリエイター、そしてプライバシーに配慮した自前のPDF処理環境を構築したいエンジニアに強く推奨されます。

注意点

ブラウザ内ですべての処理を行うため、非常に巨大なファイルの処理や、変換元の形式によっては端末のメモリを大量に消費する場合があります。オープンソースプロジェクト（AGPL-3.0ライセンス）であるため、商用利用の際はライセンス条件の確認が必要です。

まとめ

PDFCraftは、「PDFでやりたいこと」のほぼすべてを安全かつ無料で提供する、まさに決定版と言えるツールです。サーバーレスでこれほど高度な機能を実現している点は、技術的にも非常に興味深いプロジェクトです。
