これは何か

TabPFN (Tabular Prior-Data Fitted Network) は、表形式データのために特別に設計された画期的な基盤モデル（Foundation Model）です。従来の機械学習手法とは異なり、Transformer アーキテクチャを採用し、合成データを用いた膨大な事前学習によって「表形式データをどう解釈すべきか」を学習済みです。これにより、ユーザーはトレーニングデータをモデルに入力するだけで、追加のトレーニング（微調整）なしに即座に高度な予測（分類・回帰）を行うことができます。

何ができるか

CSV やデータフレーム形式の小〜中規模データセット（最大 50,000 行程度）に対して、以下のことが可能です。
・バイナリ分類、多クラス分類、および回帰タスクの実行。
・欠損値を含むデータの直接処理。
・GPU を活用した超高速な推論（1秒以内での予測完了）。
・SHAP 等を用いた予測の解釈性分析（拡張機能経由）。

目立つポイント

最大の強みは「データ前処理がほぼ不要」であることです。通常、表形式データの機械学習ではスケーリング、ワンホットエンコーディング、特徴量選択などの煩雑なパイプライン構築が必要ですが、TabPFN はそのままの数値を入力するだけで動作します。また、勾配ブースティング（XGBoost, LightGBM等）に匹敵するか、小規模データにおいてはそれらを凌駕する精度を、はるかに短い時間で達成できる点が注目されています。

セットアップや使い方の流れ

Python 3.9 以上が必要です。pip で pip install tabpfn を実行してインストールします。使い方は scikit-learn ライクで、TabPFNClassifier や TabPFNRegressor をインスタンス化し、fit(X, y) でデータを読み込ませ、predict(X_test) で予測を得るだけです。初回実行時にモデルの重み（チェックポイント）が自動的にダウンロードされます。

どんな人向けか

・データサイエンティストや研究者：ベースラインモデルの構築や、前処理に時間をかけたくない迅速なプロトタイピングに。
・機械学習を業務に取り入れたい開発者：複雑なパイプライン構築なしに、信頼性の高い分類・回帰モデルをアプリに組み込みたい場合に。
・小規模な実験データを扱うアナリスト：統計的な予測を最小限の手間で高精度に行いたい場合に最適です。

注意点

100,000 行を超えるような大規模データセットは、現時点ではメモリ消費や計算コストの観点から推奨されていません（50,000行以下が最適）。また、CPU でも動作しますが、本来のパフォーマンスを発揮するためには NVIDIA GPU が推奨されます。商用利用にはライセンスの確認が必要です（TabPFN-2.5/2.6 の重みは非商用ライセンスがデフォルトです）。

まとめ

TabPFN は、言語モデルにおける GPT のような衝撃を、表形式データの世界にもたらそうとしています。事前学習済みの知見を活かし、前処理や長いトレーニング時間をスキップして「いきなり高精度な予測」が得られる体験は、多くのデータ分析現場のワークフローを劇的に変える可能性を秘めています。
