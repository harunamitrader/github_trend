これは何か

TradingAgents は、LLM（大規模言語モデル）を搭載した複数のエージェントが、金融取引において異なる役割（アナリスト、ストラテジスト、トレーダー等）を担い、自律的に連携するフレームワークです。市場データ、ニュース、SNS の感情分析などを総合的に判断し、最適なトレード戦略を導き出します。

注目のポイント

  • 役割ベースの連携: 各エージェントに専門性を持たせることで、多角的な視点からリスク評価とリターン予測を行います。
  • リアルタイム適応: 急激な市場変動に対し、エージェント間での「会議」を通じて戦略を動的に修正する機能を備えています。
  • 多様なデータソース: 株価などの数値データだけでなく、テキストベースの情報（決算短信、X/Twitter 等）を高度に処理します。

筆者の視点

「AI によるトレード」が数値予測（RNN や Transformer）から、より高次な「推論と判断」へとシフトしている象徴的なプロジェクトです。6万スターを超える驚異的な注目度は、個人投資家からプロフェッショナルまで、エージェントによる自動運用の可能性に大きな期待を寄せていることを示しています。
