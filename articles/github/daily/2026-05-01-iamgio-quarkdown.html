これは何か

従来の Markdown の構文を拡張し、PDF やスライドなどの多様なフォーマットへの出力に対応したマークアップ言語およびコンパイラです。

何ができるか

数式、グラフ、高度なレイアウトを含む美しいドキュメントを、シンプルなテキスト形式から生成できます。

目立つポイント

Kotlin で実装されており、カスタマイズ可能なテーマや柔軟な拡張機能を提供します。

セットアップや使い方の流れ

CLI ツールをインストールし、拡張子が .qd のファイルを作成してコンパイルを実行します。

どんな人向けか

技術文書や論文、プレゼンテーション資料をテキストベースで効率よく作成したい人。

注意点

通常の Markdown にない独自記法が含まれるため、多少の学習コストがかかります。

まとめ

ドキュメント作成の表現力を大幅に引き上げる、強力な Markdown スーパーセットです。
