これは何か

Nous Research の Hermes エージェントが、Codex や Claude Code といった他の AI エージェントに対して、適切にタスクを委譲・調整するためのワークフローと命令セット（スキル）です。

何ができるか

エンジニアリングワークフローにおいて、実装は Codex、リスクレビューは Claude Code、最終統合は Hermes といった具合に、エージェントごとの特性を活かした役割分担と自律的な指揮が可能になります。

目立つポイント

明確なエージェントルーティング、親コンテキストを汚染しないコンテキスト隔離、そしてタイムアウトや出力制限による安全制御が組み込まれている点が特徴です。

セットアップや使い方の流れ

Hermes エージェントのスキルディレクトリに `SKILL.md` を追加し、設定ファイルで委譲先の外部エージェント（Codex 等）を定義することで利用可能になります。

どんな人向けか

単一のエージェントでは完結しない複雑な開発・検証タスクを、複数の AI エージェントを組み合わせて自動化したいパワーユーザーや開発者に最適です。

注意点

委譲先の外部エージェント（Codex や Claude Code）が適切にインストール・設定されており、Hermes から呼び出し可能な状態である必要があります。

まとめ

「エージェントのオーケストレーション」。複数の知能を束ねて、より大規模で複雑な課題を解決するための実用的な拡張スキルです。
