これは何か

Hermes Agent の強力な拡張機能である Kanban プラグインは、AI エージェントと人間がひとつのボード上で協力してプロジェクトを進めるためのタスク管理システムです。Linear 風のモダンな UI を提供し、タスクの割り当て、実行、監視をシームレスに行えます。

何ができるか

単なるタスクの可視化にとどまらず、「Dispatcher」機能によって「Ready」状態のタスクをエージェントが自動的に検知・実行します。進捗は「Running」「Blocked」「Done」などのステータスで管理され、エージェントが自律的にコメントを残したり、依存関係に基づいて次のタスクへ移行したりすることが可能です。

目立つポイント

SQLite（WALモード）を基盤とした堅牢なデータ管理と、エージェントのハートビート監視、自動再試行（サーキットブレーカー）など、自律型システムに必須の機能が組み込まれています。エージェントが「道具」としてではなく、ボード上で「同僚」のように振る舞う設計思想が特徴です。

セットアップや使い方の流れ

Hermes Agent 本体をインストール後、CLI から `hermes kanban` コマンドでダッシュボードを起動できます。Next.js ベースのフロントエンドにより、ブラウザから直感的にタスクを追加・操作でき、エージェント用の API キーを設定するだけで自律実行サイクルが始まります。

どんな人向けか

複数の自律型エージェントに複雑なワークフローを任せたい開発者や、AI との共同作業を視覚的に管理したいチームに最適です。特に、大規模なタスクを分解し、確実な実行ログを保持しながら進めたいプロジェクトで真価を発揮します。

注意点

実行には適切な LLM プロバイダーの API 接続が必要です。また、Dispatcher が自動的にタスクを実行するため、破壊的な操作を含むタスクを「Ready」に置く際は、エージェントの権限範囲に注意を払う必要があります。

まとめ

カンバンという馴染みのある形式に AI の実行エンジンを融合させた、次世代のプロジェクト管理ツールです。人間が「指示」をボードに置くと、AI がそれを「実行」し、結果をボードに返す——そんな未来の働き方を体現しています。
