Cursor: AI エディタの王者がさらなる高みへ

v2.6.21 のリリースと、Composer 2 / Automations の衝撃。

記事作成日 2026-03-25
最新バージョン v2.6.21
対象 Anysphere / Cursor

v2.6.21 (2026-03-23) のハイライト

最新の v2.6.21 では、UI の微調整と安定性の向上が行われましたが、真の驚きは今月発表されたばかりの **Composer 2** です。 以前のバージョンに比べ、より広いコンテキストを正確に理解し、複雑なディレクトリ構造を一撃で構築・修正する能力は、競合ツールを一歩リードしています。

2026年3月の主要なアップデート

  • Composer 2: 正式リリース。圧倒的なコーディング能力と直感的な対話インターフェース。
  • Automations: スケジュールや GitHub イベント等をトリガーにした自動化エージェント機能。
  • Cursor ACP: JetBrains IDE（IntelliJ, PyCharm 等）との統合。VS Code 以外でも Cursor のパワーが利用可能に。
  • MCP Apps: エージェント・チャット内でのインタラクティブな UI 要素をサポート。

JetBrains との統合という新たな一手

Cursor ACP (Anysphere Connection Protocol) の発表により、Cursor の恩恵は VS Code ベースの製品だけに留まらなくなりました。 JetBrains ユーザーにとっても、使い慣れた環境で Composer 2 のパワーを使えるようになったことは非常に大きなニュースです。

まとめ

Cursor は、単に LLM を呼び出すエディタではなく、IDE 自体を「AI のために再定義」し続けています。 Automations による自動化エージェントの流れは、今後の開発現場の在り方を根本から変える可能性を秘めています。
